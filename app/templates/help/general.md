# SEO Management Platform

Внутренняя платформа для управления SEO-продвижением WordPress-сайтов.

## Модули системы

| Модуль | Что делает |
|--------|-----------|
| **Sites** | Управление сайтами, WP-подключение, группы |
| **Keywords** | Ключевые слова, группы, импорт из файлов |
| **Positions** | Отслеживание позиций, графики, расписания |
| **Crawl** | Краулинг страниц, SEO-аудит, change feed |
| **Clusters** | Кластеризация ключей, каннибализация |
| **Pipeline** | Контент-пайплайн: TOC, schema.org, внутренние ссылки |
| **Projects** | Проекты, Kanban-доска, контент-план, брифы |
| **Reports** | Дашборд, Excel-отчёты, ad-трафик |
| **Admin** | Пользователи, роли, группы доступа, аудит |

## Как пользоваться справкой

Иконка **?** в заголовке каждой страницы открывает справку по текущему модулю.

Справки описывают:
- Что умеет модуль
- Как выполнить основные операции
- Какие есть ограничения
- Связи с другими модулями

Закрыть справку: кнопка **&times;**, клик вне окна, или клавиша **Esc**.
