# Отчёты и аналитика

Дашборд, сводка по сайтам, Excel-экспорт, загрузка рекламного трафика.

## Dashboard

Главная страница платформы (`/ui/dashboard`):

### Общая статистика
- Количество сайтов, проектов, ключевых слов
- Открытые задачи, задачи в работе

### Sites Overview
Таблица по всем сайтам:
- Количество ключей
- Распределение позиций: TOP-3, TOP-10, TOP-30
- Открытые задачи
- Быстрые ссылки на позиции и ключи

### Проекты
Последние 20 проектов с Kanban и контент-планом.

## Site Overview API

`GET /reports/sites/{id}/overview` — сводка по одному сайту:
- Количество ключей, задач, краулов
- Распределение позиций (TOP-3/10/30/100)
- Top movers (самые большие изменения за 7 дней):
  - **Gainers** — ключи с наибольшим ростом позиций
  - **Losers** — ключи с наибольшим падением

## Excel-экспорт

`GET /reports/projects/{id}/excel` — скачать отчёт по проекту:
- Лист «Positions» — ключи с позициями и дельтами
- Лист «Keywords» — фразы, частотность, регионы
- Лист «Tasks» — задачи с типами и статусами

## Рекламный трафик

### Загрузка CSV
`POST /reports/sites/{id}/ad-traffic/upload` — загрузить CSV с рекламным трафиком.
Формат: `source,date,sessions,conversions,cost`

### Сравнение периодов
`POST /reports/sites/{id}/ad-traffic/compare` — сравнить два периода по источникам трафика:
- Сессии, конверсии, расходы
- Процент изменения

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/reports/dashboard` | Общая статистика |
| GET | `/reports/sites/{id}/overview` | Сводка по сайту |
| GET | `/reports/projects/{id}/excel` | Excel-экспорт |
| POST | `/reports/sites/{id}/ad-traffic/upload` | Загрузка CSV |
| POST | `/reports/sites/{id}/ad-traffic/compare` | Сравнение периодов |

## Связи с другими модулями

- **Sites** — сводка по каждому сайту
- **Positions** — распределение, top movers
- **Keywords** — подсчёт, экспорт
- **Tasks** — подсчёт открытых
- **Projects** — Excel-экспорт по проекту
