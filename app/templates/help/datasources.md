# Источники данных

Модуль интеграции с внешними сервисами: Google Search Console, Yandex Webmaster, SERP-парсер.

## Подключённые источники

### Google Search Console (GSC)
- **Данные:** клики, показы, CTR, средняя позиция по запросам
- **Подключение:** OAuth 2.0 (кнопка на странице сайта)
- **Обновление:** по запросу или по расписанию
- Токены хранятся зашифрованными (Fernet), автообновление при истечении

### Yandex Webmaster
- **Данные:** запросы, показы, клики, позиции в Яндексе
- **Подключение:** OAuth-токен в переменной окружения `YANDEX_WEBMASTER_TOKEN`
- **Обновление:** по запросу

### SERP Parser (Playwright)
- **Данные:** органическая выдача Google и Яндекс (TOP-20)
- **Лимит:** 50 запросов в день (настраивается)
- **Детект SERP-фич:** featured snippet, People Also Ask, видео, картинки, knowledge panel, local pack, реклама
- Не требует API-ключей, работает через headless-браузер

### Yandex Metrika (планируется)
- Органический трафик на страницу
- Визиты, глубина просмотра, время на сайте

## SERP-фичи
При парсинге выдачи определяются:
- **Featured Snippet** — расширенный ответ в выдаче
- **PAA** (People Also Ask) — блок «Похожие вопросы»
- **Video** — видеокарусель
- **Images** — блок картинок
- **Knowledge Panel** — информационная панель
- **Local Pack** — карта с локальными результатами
- **Ads** — рекламные объявления

## Страница Data Sources
Доступна по `/ui/datasources`. Показывает статус подключений и использование SERP-парсера.

## API-эндпоинты

### GSC
| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/gsc/authorize/{site_id}` | URL для OAuth |
| GET | `/gsc/callback` | OAuth callback |
| POST | `/gsc/{site_id}/fetch` | Загрузить данные |

### Yandex
| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/yandex/status` | Статус подключения |
| GET | `/yandex/hosts` | Список хостов |
| POST | `/yandex/queries` | Загрузить запросы |
| GET | `/yandex/serp/usage` | Использование SERP |

## Связи с другими модулями
- **Positions** — данные GSC/Yandex записываются как позиции
- **Clusters** — SERP-данные используются для auto-clustering
- **Keywords** — частотность может подтягиваться из внешних источников
