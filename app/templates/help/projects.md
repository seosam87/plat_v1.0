# Проекты и задачи

Модуль управления SEO-проектами: Kanban-доска задач, контент-план, брифы, комментарии.

## Проекты

### Создание проекта
Каждый проект привязан к сайту. Поля: название, описание, клиент (опционально).

### Статусы проекта
- **active** — в работе
- **paused** — приостановлен
- **completed** — завершён
- **archived** — архивирован

### Доступ
Администратор видит все проекты. Manager/Client — только проекты, к которым назначен доступ.

## Задачи (SEO Tasks)

### Типы задач
- **page_404** — страница возвращает 404 (авто)
- **lost_indexation** — страница получила noindex (авто)
- **missing_page** — ключ без целевой страницы (авто)
- **cannibalization** — каннибализация ключей (авто)
- **manual** — ручная задача

### Приоритеты
- **P1** — критический (красный)
- **P2** — высокий (жёлтый)
- **P3** — средний (серый, по умолчанию)
- **P4** — низкий

### Kanban-доска
5 колонок: Open → Assigned → In Progress → Review → Resolved.
Смена статуса — кнопками в карточке задачи.

### Фильтры задач
Страница `/ui/tasks` — фильтрация по:
- **Статус** — open, assigned, in_progress, review, resolved
- **Сайт** — выбор из списка
- **Тип** — page_404, lost_indexation, manual и т.д.
- **Приоритет** — P1, P2, P3, P4

### Виджет задач на странице сайта
На странице детали сайта — блок «Open Tasks» с 5 последними задачами и ссылкой на полный список.

## Контент-план

Таблица с планируемыми статьями/страницами:
- Название, статус (idea → planned → writing → review → published)
- Дата публикации
- Создание WP-черновика одной кнопкой

## Брифы

Генерация брифа из кластера ключевых слов:
- H1 из основного ключа
- H2 из топ-5 ключей
- Таблица ключей с частотностью
- Скачиваемый HTML

## Комментарии
К каждому проекту можно добавлять комментарии.

## Связи с другими модулями
- **Sites** — проект привязан к сайту
- **Crawl** — автозадачи из результатов краулинга
- **Clusters** — брифы из кластеров, missing page detection
- **Pipeline** — контент-план → WP draft
