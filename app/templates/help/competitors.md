# Конкуренты

Модуль отслеживания конкурентов: список доменов, сравнение позиций, автоматическое обнаружение из SERP.

## Что можно делать

### Добавление конкурентов
Ручное добавление: введите домен (example.com) и опциональное название. Протокол (https://) и www удаляются автоматически.

### Авто-обнаружение
Кнопка **Auto-Detect** анализирует SERP-данные и находит домены, которые ранжируются по тем же ключам в TOP-10:
- Показывает количество общих ключей
- Среднюю позицию конкурента
- Ваш домен исключается из результатов

### Сравнение позиций
Кнопка **Compare** — модальное окно с таблицей:
- Ключевое слово
- Наша позиция vs позиция конкурента
- Delta: кто впереди и на сколько

Данные берутся из `keyword_positions` — URL ранжирования сопоставляется с доменом конкурента.

### Удаление
Кнопка × удаляет конкурента с подтверждением.

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|---------|
| POST | `/competitors/sites/{id}` | Добавить конкурента |
| GET | `/competitors/sites/{id}` | Список конкурентов |
| DELETE | `/competitors/{id}` | Удалить |
| GET | `/competitors/sites/{id}/compare/{comp_id}` | Сравнение позиций |
| GET | `/competitors/sites/{id}/detect` | Авто-обнаружение из SERP |

## Ограничения текущей версии

- Сравнение работает только по данным из нашего SERP-парсера (конкурент должен быть в выдаче)
- Нет отдельного парсинга позиций конкурента — только по пересечениям
- Нет сравнения SEO-полей страниц (планируется)
- Нет отслеживания backlinks конкурента

## Связи с другими модулями

- **Sites** — конкуренты привязаны к сайту
- **Positions** — данные для сравнения берутся из keyword_positions
- **Clusters** — SERP-пересечения используются для обнаружения
