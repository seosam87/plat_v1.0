## Critical Integration Gaps

### INT-01: /m/sites route missing (MOB-01, HLT-01, HLT-02)

`base_mobile.html` line 73 has bottom nav "Сайты" tab linking to `/m/sites`. No such route exists in `mobile.py`. Clicking returns 404.

**Fix:** Add `GET /m/sites` route — site picker page that links to `/m/health/{site_id}`.

### INT-02: /m/reports route missing (REP-01, REP-02, BOT-03)

Bot `miniapp.py:54` and `seo.py:144,204` create Mini App buttons to `/m/reports`. Only `/m/reports/new` exists.

**Fix:** Add `GET /m/reports` redirect to `/m/reports/new`.

### INT-03: /m/health without site_id (BOT-03, HLT-01)

Bot `miniapp.py:59` creates "Здоровье" button to `/m/health` (no site_id). Router requires `{site_id}`.

**Fix:** Change bot button to `/m/sites` or add `/m/health` site picker.

### INT-04: Template extends path broken (TLS-01, REP-01)

4 templates (`mobile/reports/new.html`, `mobile/tools/list.html`, `mobile/tools/result.html`, `mobile/tools/run.html`) extend `mobile/base_mobile.html`. File `app/templates/mobile/base_mobile.html` does not exist — only `app/templates/base_mobile.html`.

**Fix:** Change `{% extends "mobile/base_mobile.html" %}` to `{% extends "base_mobile.html" %}`.

### INT-05: showToast HX-Trigger not wired (PAG-03, PAG-04)

`base_mobile.html` defines `showToast()` function but has no `htmx:afterOnLoad` listener. Quick fix and bulk operation toasts silently fail. Green confirmation div still shows (partial feedback).

**Fix:** Add HTMX event listener matching `pipeline/index.html` pattern.

### INT-06: Digest links go to desktop routes (DIG-02)

All 4 digest section links point to `/ui/*` desktop routes instead of `/m/*` mobile routes.

**Fix:** Update to `/m/health/{site_id}`, `/m/errors`, `/m/positions`, `/m/errors`.
