<action>
Create `app/models/metrika.py` with three SQLAlchemy models:

**MetrikaTrafficDaily** — daily site-aggregate for chart and widget:
- `id`: UUID PK, default uuid4
- `site_id`: UUID FK → sites.id ON DELETE CASCADE, not null
- `traffic_date`: Date, not null
- `visits`: Integer, not null, default 0
- `users`: Integer, not null, default 0
- `bounce_rate`: Numeric(5,2), nullable
- `page_depth`: Numeric(5,2), nullable
- `avg_duration_seconds`: Integer, nullable
- `created_at`: DateTime(timezone=True), default utcnow
- `__table_args__`: UniqueConstraint('site_id', 'traffic_date', name='uq_metrika_daily_site_date')

**MetrikaTrafficPage** — per-page aggregate over a date range:
- `id`: UUID PK, default uuid4
- `site_id`: UUID FK → sites.id ON DELETE CASCADE, not null
- `period_start`: Date, not null
- `period_end`: Date, not null
- `page_url`: Text, not null
- `visits`: Integer, not null, default 0
- `bounce_rate`: Numeric(5,2), nullable
- `page_depth`: Numeric(5,2), nullable
- `avg_duration_seconds`: Integer, nullable
- `created_at`: DateTime(timezone=True), default utcnow
- `__table_args__`: UniqueConstraint('site_id', 'period_start', 'period_end', 'page_url', name='uq_metrika_pages_site_period_url')

**MetrikaEvent** — manual event markers for chart overlay:
- `id`: UUID PK, default uuid4
- `site_id`: UUID FK → sites.id ON DELETE CASCADE, not null
- `event_date`: Date, not null
- `label`: String(255), not null
- `color`: String(20), default '#6b7280'
- `created_at`: DateTime(timezone=True), default utcnow

Import pattern follows `app/models/ad_traffic.py`:
```python
import uuid
from datetime import date, datetime

from sqlalchemy import Date, DateTime, ForeignKey, Integer, Numeric, String, Text, UniqueConstraint
from sqlalchemy.dialects.postgresql import UUID
from sqlalchemy.orm import Mapped, mapped_column

from app.database import Base
```

Table names: `metrika_traffic_daily`, `metrika_traffic_pages`, `metrika_events`.
