**Bug 5 — Run Batch feedback:**

The "Run Batch" button currently uses `hx-swap="none"` so the 202 JSON response is discarded silently. Fix:

1. Add a result container div after the buttons div (after the `<div class="flex gap-2">` block, before the jobs table):
```html
<div id="batch-result" class="hidden mb-4 p-3 rounded-lg bg-blue-50 border border-blue-200 text-sm text-blue-800"></div>
```

2. Update the "Run Batch" button to use `hx-target="#batch-result"` and `hx-swap="innerHTML"` instead of `hx-swap="none"`. Remove the `onclick` handler that disabled the button (we'll handle state differently). The updated button:
```html
<button id="btn-batch" class="btn btn-primary"
  hx-post="/pipeline/sites/{{ site_id }}/batch"
  hx-target="#batch-result"
  hx-swap="innerHTML"
  hx-indicator="#batch-spinner"
  hx-on::before-request="this.disabled=true; this.textContent='Running...';"
  hx-on::after-request="this.disabled=false; this.textContent='Run Batch';">Run Batch</button>
```

3. Add a script block (inside the existing `<script>` tag at the bottom) that listens for the HTMX response and formats it:
```javascript
document.body.addEventListener('htmx:afterRequest', function(evt) {
  if (evt.detail.target && evt.detail.target.id === 'batch-result') {
    var el = document.getElementById('batch-result');
    el.classList.remove('hidden');
    if (evt.detail.successful) {
      try {
        var data = JSON.parse(evt.detail.xhr.responseText);
        if (data.jobs_created > 0) {
          el.className = 'mb-4 p-3 rounded-lg bg-emerald-50 border border-emerald-200 text-sm text-emerald-800';
          el.textContent = data.jobs_created + ' pipeline job(s) created. Refresh the page in a moment to see results.';
        } else {
          el.className = 'mb-4 p-3 rounded-lg bg-amber-50 border border-amber-200 text-sm text-amber-800';
          el.textContent = 'No pages need enrichment. All pages already have TOC and schema.org markup.';
        }
      } catch(e) {
        el.className = 'mb-4 p-3 rounded-lg bg-emerald-50 border border-emerald-200 text-sm text-emerald-800';
        el.textContent = 'Batch pipeline started. Refresh the page in a moment to see results.';
      }
    } else {
      el.className = 'mb-4 p-3 rounded-lg bg-red-50 border border-red-200 text-sm text-red-800';
      el.textContent = 'Batch failed: ' + (evt.detail.xhr.status || 'unknown error') + '. Check that the site has crawled pages.';
    }
  }
});
```
