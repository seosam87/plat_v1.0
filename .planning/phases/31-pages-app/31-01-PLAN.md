**2. Create GET /m/pages endpoint in mobile.py** (per D-01, D-02, D-03, D-06, D-07):

Add `async def mobile_pages(request: Request, db: AsyncSession = Depends(get_db), current_user = Depends(get_current_user))` at the end of mobile.py (before the last line).

Route: `@mobile_router.get("/m/pages")`

Logic:
- Get all sites for user: `sites = (await db.execute(select(Site).order_by(Site.domain))).scalars().all()`
- If no sites: render index.html with `sites=[], pages=[], active_tab='pages'`
- Get site_id from `request.query_params.get("site_id")` or `request.cookies.get("m_pages_site_id")`, fallback to `str(sites[0].id)`
- Get active tab from `request.query_params.get("tab", "all")`
- Get offset from `int(request.query_params.get("offset", "0"))`
- Build latest crawl subquery:
  ```python
  from sqlalchemy import select, func
  from app.models.crawl import CrawlJob, CrawlJobStatus, Page
  latest_crawl_sq = (
      select(CrawlJob.id)
      .where(CrawlJob.site_id == uuid.UUID(site_id), CrawlJob.status == CrawlJobStatus.done)
      .order_by(CrawlJob.finished_at.desc())
      .limit(1)
      .scalar_subquery()
  )
  ```
- Build base query: `base_q = select(Page).where(Page.crawl_job_id == latest_crawl_sq)`
- Count queries for all 4 tabs (execute 4 `select(func.count()).where(...)` queries):
  - `count_all`: no extra filter
  - `count_no_schema`: `.where(Page.has_schema == False)`
  - `count_no_toc`: `.where(Page.has_toc == False)`
  - `count_noindex`: `.where(Page.has_noindex == True)`
- Apply tab filter to pages query based on `tab` param
- Order by `Page.url`, `.limit(21).offset(offset)` (fetch 21 to detect "has more")
- Execute query, take first 20 results, set `has_more = len(rows) > 20`
- Build context: `sites`, `pages`, `site_id`, `tab`, `counts={"all": count_all, "no_schema": count_no_schema, "no_toc": count_no_toc, "noindex": count_noindex}`, `has_more`, `offset`, `next_offset=offset+20`, `active_tab='pages'`
- If `request.headers.get("HX-Request")`: render `mobile/pages/partials/pages_content.html`
- Else: render `mobile/pages/index.html`
- Set cookie: `response.set_cookie(key="m_pages_site_id", value=site_id, httponly=True, samesite="lax", max_age=86400*30)`
