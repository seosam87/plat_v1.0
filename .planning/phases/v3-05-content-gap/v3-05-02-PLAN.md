### Pure scoring function

**`compute_potential_score(frequency: int | None, competitor_position: int | None) -> float`**
```python
if not frequency or not competitor_position:
    return 0.0
if competitor_position <= 3:
    factor = 1.0
elif competitor_position <= 10:
    factor = 0.7
elif competitor_position <= 30:
    factor = 0.3
else:
    factor = 0.1
return round(frequency * factor, 1)
```

**`SCORE_FORMULA_DESCRIPTION`** — constant string:
```
"Потенциал = Частотность × Коэффициент позиции конкурента. Коэффициент: TOP-3 = 1.0, TOP-10 = 0.7, TOP-30 = 0.3, остальные = 0.1. Чем выше значение — тем перспективнее ключ для создания контента."
```

### Gap detection from SERP (async)

**`async def detect_gaps_from_session(db, site_id, session_id, competitor_domain) -> list[dict]`**
1. Load SessionSerpResults for the session
2. Load our keywords for the site (phrase set)
3. For each SERP result where competitor_domain appears but our domain doesn't appear in TOP-10:
   - This is a gap — competitor ranks, we don't
4. Return list of `{phrase, frequency, competitor_position, our_position, potential_score}`

**`async def save_gap_keywords(db, site_id, competitor_domain, gaps: list[dict], source='serp') -> int`**
Upsert gap keywords. Compute potential_score for each. Return count saved.

### Import from file (async)

**`async def import_competitor_keywords(db, site_id, competitor_domain, file_path: str) -> dict`**
1. Use `base.read_file()` to parse CSV/XLSX
2. Use `gap_parser.parse_gap_file()` to extract {phrase, frequency, position}
3. Compare with our keywords — find gaps
4. Save as GapKeyword with source='csv_import' or 'xlsx_import'
5. Return `{imported: N, gaps_found: N}`

### Group management (async)

### Gap keyword CRUD (async)

**`async def list_gap_keywords(db, site_id, competitor_domain=None, group_id=None, min_score=None, limit=500, offset=0) -> tuple[list[dict], int]`**

### Proposal management (async)

**`async def create_proposals_from_gaps(db, site_id, keyword_ids: list[UUID]) -> list[GapProposal]`**
For each keyword: create a GapProposal with title="Написать текст: {phrase}", target_phrase=phrase, frequency, potential_score. Status=pending.

**`async def approve_proposal(db, proposal_id, project_id=None) -> GapProposal`**
Set status=approved. Optionally create ContentPlanItem in the given project.

**`async def reject_proposal(db, proposal_id) -> GapProposal`**
Set status=rejected.
