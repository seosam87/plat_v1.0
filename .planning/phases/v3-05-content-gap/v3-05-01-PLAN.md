</objective>

<title>Create app/models/gap.py with three models</title>
<read_first>
- app/models/keyword.py

</read_first>
<action>
Create `app/models/gap.py`:

**GapKeyword** — competitor keyword not in our set:
- `id`: UUID PK
- `site_id`: UUID FK → sites.id CASCADE
- `competitor_domain`: String(500), not null
- `phrase`: String(1000), not null
- `frequency`: Integer, nullable
- `competitor_position`: Integer, nullable — competitor's position for this keyword
- `our_position`: Integer, nullable — our position (null = not ranking)
- `potential_score`: Float, nullable — computed score
- `gap_group_id`: UUID FK → gap_groups.id SET NULL, nullable
- `source`: String(50), default 'serp' — 'serp', 'csv_import', 'xlsx_import'
- `created_at`: DateTime(timezone=True)
- `__table_args__`: UniqueConstraint('site_id', 'competitor_domain', 'phrase', name='uq_gap_keyword_site_comp_phrase')

**GapGroup** — manual grouping:
- `id`: UUID PK
- `site_id`: UUID FK → sites.id CASCADE
- `name`: String(255), not null
- `created_at`: DateTime(timezone=True)

**ProposalStatus** enum: `pending`, `approved`, `rejected`

**GapProposal** — content plan proposal:
- `id`: UUID PK
- `site_id`: UUID FK → sites.id CASCADE
- `gap_keyword_id`: UUID FK → gap_keywords.id CASCADE, nullable — source gap keyword
- `title`: String(500), not null — proposed content title
- `target_phrase`: String(1000), not null — main keyword phrase
- `frequency`: Integer, nullable
- `potential_score`: Float, nullable
- `status`: SAEnum(ProposalStatus), default `pending`
- `notes`: Text, nullable
- `content_plan_item_id`: UUID FK → content_plan_items.id SET NULL, nullable — linked after approval
- `created_at`: DateTime(timezone=True)
- `updated_at`: DateTime(timezone=True), onupdate

Table names: `gap_keywords`, `gap_groups`, `gap_proposals`.
</action>
