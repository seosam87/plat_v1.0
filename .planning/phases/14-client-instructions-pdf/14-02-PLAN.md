**1. Create `app/tasks/client_report_tasks.py`:**

```python
"""Celery tasks for client instruction PDF generation and delivery."""
from __future__ import annotations
import asyncio
from loguru import logger
from app.celery_app import celery_app

@celery_app.task(
    name="app.tasks.client_report_tasks.generate_client_pdf",
    bind=True,
    max_retries=1,
    queue="default",
    soft_time_limit=90,
    time_limit=120,
)
def generate_client_pdf(self, report_id: str, site_id: str, blocks_config: dict) -> dict:
    """Generate client instruction PDF in subprocess-isolated WeasyPrint."""
    import uuid
    from app.database import AsyncSessionLocal
    from app.services.client_report_service import (
        generate_client_report, save_report_pdf, mark_report_failed,
    )

    async def _run():
        async with AsyncSessionLocal() as db:
            try:
                pdf_bytes = await generate_client_report(db, uuid.UUID(site_id), blocks_config)
                await save_report_pdf(db, uuid.UUID(report_id), pdf_bytes)
                await db.commit()
                logger.info("Client PDF generated", report_id=report_id, size=len(pdf_bytes))
                return {"status": "ready", "size": len(pdf_bytes)}
            except Exception as exc:
                await db.rollback()
                await mark_report_failed(db, uuid.UUID(report_id), str(exc))
                await db.commit()
                logger.error("Client PDF generation failed", report_id=report_id, error=str(exc))
                raise

    try:
        return asyncio.run(_run())
    except Exception as exc:
        logger.error("Client PDF task failed", error=str(exc))
        raise self.retry(exc=exc, countdown=10)
```

Endpoints:

a) `GET /ui/client-reports/` — main page (response_class=HTMLResponse):
   - Query param `site_id` (optional UUID, default from session/cookie or first site)
   - Load site list (all sites), load report history for selected site
   - Return `templates.TemplateResponse(request, "client_reports/index.html", {"sites": sites, "site": selected_site, "reports": history})`

b) `POST /ui/client-reports/generate` — start generation (HTMX):
   - Form fields: `site_id` (UUID), `quick_wins` (bool), `audit_errors` (bool), `dead_content` (bool), `positions` (bool)
   - Build `blocks_config = {"quick_wins": quick_wins, "audit_errors": audit_errors, "dead_content": dead_content, "positions": positions}`
   - Validate at least one block selected
   - Call `create_report_record(db, site_id, blocks_config)`, get report back
   - Dispatch `generate_client_pdf.delay(str(report.id), str(site_id), blocks_config)`
   - Update report.celery_task_id = task.id, commit
   - Return HTML partial `client_reports/partials/generation_status.html` with status="generating", report_id, `hx-get` polling URL

c) `GET /ui/client-reports/status/{report_id}` — poll status (HTMX):
   - Load ClientReport by id
   - If status=="ready": return generation_status.html with download link + delivery buttons, set `HX-Trigger: refreshHistory` header
   - If status=="failed": return generation_status.html with error message
   - If status in ("pending","generating"): return generation_status.html with spinner + `hx-get` self-polling every 3s via `hx-trigger="load delay:3s"`

d) `GET /ui/client-reports/{report_id}/download` — download PDF:
   - Load report, check status=="ready" and pdf_data is not None
   - Return `Response(content=report.pdf_data, media_type="application/pdf", headers={"Content-Disposition": f'attachment; filename="client-report-{report.id}.pdf"'})`

e) `POST /ui/client-reports/{report_id}/send-email` — send via email (HTMX):
   - Dispatch `send_client_report_email.delay(str(report_id))`
   - Return toast HTML: `<div class="toast toast-success">Отчёт отправлен на email.</div>`

f) `POST /ui/client-reports/{report_id}/send-telegram` — send via Telegram (HTMX):
   - Dispatch `send_client_report_telegram.delay(str(report_id))`
   - Return toast HTML: `<div class="toast toast-success">Отчёт отправлен в Telegram.</div>`

g) `GET /ui/client-reports/history` — history table partial (HTMX):
   - Query param `site_id` (UUID)
   - Return `client_reports/partials/history_table.html`
