# Phase 999.10 Plan 02: QA Surface Router and Templates Summary

**One-liner:** FastAPI router (/ui/qa/) with 11 endpoints + 4 Jinja2 templates implementing status-colored matrix view, feature CRUD, HTMX inline check editor, and route candidate discovery page.

## What Was Built

**Task 1: Router + wiring + smoke prefix**

Created `app/routers/qa_surfaces.py` with 11 endpoints:
- `GET /ui/qa/` — matrix view (features × surfaces)
- `GET /ui/qa/features/new` — create feature form
- `POST /ui/qa/features` — create feature
- `GET /ui/qa/features/{id}` — edit feature form
- `POST /ui/qa/features/{id}` — update feature
- `POST /ui/qa/features/{id}/delete` — delete feature (cascade)
- `GET /ui/qa/checks/{id}/edit` — HTMX partial: edit mode
- `POST /ui/qa/checks/{id}/mark-tested` — mark check tested
- `GET /ui/qa/checks/{id}` — HTMX partial: view mode (cancel restore)
- `GET /ui/qa/candidates` — route discovery page (deferred import pattern)
- `POST /ui/qa/candidates/save` — save selected routes as user flow

Router wired into `app/main.py`, QA Surfaces added as first child of settings nav section, `/ui/qa` added to `_HELP_MODULE_MAP`, `/ui/qa` added to `UI_PREFIXES`.

**Task 2: Jinja2 templates**

Created `app/templates/qa/` directory with 4 templates:
- `index.html` — full matrix table with status-colored cells (HTMX `hx-get` on click), empty state with "No user flows registered yet", "Add User Flow" CTA, "Discover Routes" link
- `feature_form.html` — create/edit form with name, slug, description, retest_days fields; auto-slug JS from name input; conditional "Create User Flow" / "Update User Flow" submit; "Back to Matrix" secondary link
- `_check_cell.html` — HTMX partial (no base.html extend); edit mode with status dropdown + notes textarea + "Mark as Tested" button (`hx-disabled-elt="this"`) + cancel link; view mode with STATUS_COLORS badge and SVG icons for `needs_retest` (clock) and `failed` (X)
- `candidates.html` — routes grouped by prefix with checkboxes, right-side flow definition form, auto-slug JS, "Add User Flow" submit

All copy matches UI-SPEC copywriting contract exactly.
