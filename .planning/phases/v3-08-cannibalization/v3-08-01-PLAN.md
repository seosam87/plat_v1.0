<task id="02">
<title>Create cannibalization_service.py</title>
<read_first>
- app/services/cluster_service.py (detect_cannibalization)
- app/models/cannibalization.py
- app/models/task.py
</read_first>
<action>
Create `app/services/cannibalization_service.py`:

**`def suggest_resolution_type(competing_urls: list[str], keyword: str) -> str`** (pure)
Heuristic: if URLs are similar (same path prefix) → merge_content. If one has much higher inlinks → set_canonical. If one is a redirect target → redirect_301. Default → split_keywords.

**`def generate_action_plan(resolution_type: str, keyword: str, competing_urls: list[str], primary_url: str) -> str`** (pure)
Generate text action plan based on type:
- merge: "Объединить контент страниц X и Y. Перенести уникальный контент на {primary_url}. Настроить 301 с удалённой страницы."
- canonical: "Установить canonical на {primary_url} для всех конкурирующих страниц."
- redirect: "Настроить 301 redirect с {other_urls} на {primary_url}."
- split: "Переназначить ключ '{keyword}' на {primary_url}. Для остальных страниц подобрать другие ключевые слова."

**Async CRUD:**
- `create_resolution(db, site_id, keyword, urls, type, primary_url) -> CannibalizationResolution`
- `create_resolution_task(db, resolution_id) -> SeoTask` — creates SeoTask with type=cannibalization
- `list_resolutions(db, site_id, status=None)`
- `update_resolution_status(db, resolution_id, status)`
- `check_resolution(db, resolution_id) -> dict` — re-run cannibalization check for this keyword, return {still_cannibalized: bool}
</action>
