**Variable parser functions:**
```python
import re

_VAR_RE = re.compile(r"\{\{(\w+)\}\}")

def parse_template_variables(template: str) -> list[str]:
    """Extract unique ordered variable names from {{var}} template."""
    seen: set[str] = set()
    result: list[str] = []
    for match in _VAR_RE.finditer(template):
        name = match.group(1)
        if name not in seen:
            seen.add(name)
            result.append(name)
    return result

def render_template(template: str, inputs: dict[str, str]) -> str:
    """Substitute {{var}} placeholders with provided values."""
    def replacer(m: re.Match) -> str:
        return inputs.get(m.group(1), m.group(0))
    return _VAR_RE.sub(replacer, template)
```

**Step 2: Add `call_agent()` to `app/services/llm/llm_service.py`.**
Add at the END of the file (do not modify existing functions):
```python
async def call_agent(
    api_key: str,
    *,
    system_prompt: str,
    user_message: str,
    model: str,
    temperature: float,
    max_tokens: int,
) -> tuple[str, int, int]:
    """Call Anthropic for agent execution. Returns (output_text, input_tokens, output_tokens)."""
    from anthropic import AsyncAnthropic
    client = AsyncAnthropic(api_key=api_key)
    response = await client.messages.create(
        model=model,
        max_tokens=max_tokens,
        system=system_prompt,
        messages=[{"role": "user", "content": user_message}],
        temperature=temperature,
    )
    text = response.content[0].text
    return text, response.usage.input_tokens, response.usage.output_tokens
```

- Task `run_agent_job`:
  - `@celery_app.task(name="agents.run_agent_job", bind=True, max_retries=3, queue="default")`
  - Sync wrapper: `asyncio.new_event_loop()` + `loop.run_until_complete(_run_agent_job(...))`
  - Catch `_TransientAgentError` → `self.retry(exc=exc, countdown=60 * 2 ** self.request.retries)`
- Async `_run_agent_job(job_id: int, current_retry: int, max_retries: int)`:
  1. Load AgentJob from DB, set status="running", commit
  2. Load AIAgent by job.agent_id (if None, set error, return)
  3. Render user_template: `render_template(agent.user_template, job.inputs_json or {})`
  4. Get API key from `settings.ANTHROPIC_API_KEY`
  5. Check circuit breaker via Redis (same pattern as llm_tasks.py `check_circuit_breaker`)
  6. Try `call_agent(api_key, system_prompt=agent.system_prompt, user_message=rendered, model=agent.model, temperature=float(agent.temperature), max_tokens=agent.max_tokens)`
  7. On success: set job.status="done", job.output_text=text, job.input_tokens, job.output_tokens, commit
  8. Call `increment_usage_count(db, agent.id)` on success
  9. On `anthropic.APIStatusError` with status 429/529/500/502/503 → raise `_TransientAgentError`
  10. On `anthropic.APIStatusError` with status 401/403/400 → set job.status="error", job.error_message, commit (permanent failure, no retry)
  11. On `anthropic.APIConnectionError` / `anthropic.APITimeoutError` → raise `_TransientAgentError`
  12. If retries exhausted (in except block after final retry fails) → set job.status="error", job.error_message="API недоступен после 3 попыток", commit
