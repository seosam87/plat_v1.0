**Step 1: Create `app/routers/agents.py`.**

```python
"""AI Agent catalogue, CRUD, execution, and HTMX partials (Phase 999.9).

Routes:
- GET  /ui/agents/                    — catalogue page (full + HTMX grid partial)
- GET  /ui/agents/parse-vars          — HTMX: parse {{vars}} from template text
- GET  /ui/agents/new                 — create form
- POST /ui/agents/new                 — create agent
- GET  /ui/agents/{id}/edit           — edit form
- POST /ui/agents/{id}/edit           — update agent
- POST /ui/agents/{id}/delete         — delete agent
- GET  /ui/agents/{id}/run            — run page
- POST /ui/agents/{id}/run            — dispatch Celery job
- GET  /ui/agents/jobs/{job_id}/status — HTMX poll job status
- POST /ui/agents/{id}/favourite      — toggle favourite (HTMX)
- POST /ui/agents/{id}/fork           — fork agent
"""
```

Router: `router = APIRouter(prefix="/ui/agents", tags=["agents"])`

**Endpoints to implement:**

**GET /ui/agents/** — catalogue page
- Query params: `category` (str, optional), `search` (str, optional), `favourite` (int, optional: 1=only favourites)
- Call `list_categories(db)` for sidebar filter
- Call `list_agents(db, category_slug=category, search=search, favourite_user_id=user.id if favourite else None)`
- For each agent, check `is_favourited(db, user_id=user.id, agent_id=agent.id)` — or batch this with a subquery in list_agents
- If request has `HX-Request` header → return `_agent_card_grid.html` partial (HTMX swap target)
- Else → return full `agents/index.html` with context: `agents`, `categories`, `current_category`, `search_query`, `favourite_mode`

**GET /ui/agents/parse-vars** — HTMX partial for variable preview
- Query param: `template` (str, default="")
- Call `parse_template_variables(template)`
- Return inline HTML: if no vars, show hint text "Переменных не обнаружено. Используйте `{{переменная}}` в шаблоне запроса."
- If vars found, render slate badges: `<span class="inline-block px-2 py-0.5 text-xs rounded bg-slate-100 text-slate-600 mr-1 mb-1">{{var_name}}</span>` per var

**GET /ui/agents/new** — create form page
- Load categories for select dropdown
- Render `agents/form.html` with empty form, `is_edit=False`

**POST /ui/agents/new** — process create form
- Extract form fields: name, description, icon, category_id, system_prompt, user_template, model, temperature (float), max_tokens (int), output_format, tags (comma-split → list), is_public (checkbox)
- Call `create_agent(db, ..., created_by=user.id)`
- Commit, redirect to `/ui/agents/` with 302

**GET /ui/agents/{agent_id}/edit** — edit form page
- Load agent via `get_agent(db, agent_id)`, 404 if not found
- Load categories for select
- Render `agents/form.html` with `agent=agent, is_edit=True`

**POST /ui/agents/{agent_id}/edit** — process edit form
- Extract same fields as create
- Call `update_agent(db, agent_id, **fields)`
- Commit, redirect to `/ui/agents/` with 302

**POST /ui/agents/{agent_id}/delete** — delete agent
- Call `delete_agent(db, agent_id)`
- Commit, redirect to `/ui/agents/` with 302

**GET /ui/agents/{agent_id}/run** — run page
- Load agent, parse variables from `agent.user_template`
- Render `agents/run.html` with `agent=agent, variables=variables`

**POST /ui/agents/{agent_id}/run** — dispatch LLM job
- Load agent
- Extract variable values from form: `{var: request.form[var] for var in parse_template_variables(agent.user_template)}`
- Create job: `create_job(db, agent_id=agent.id, agent_name=agent.name, user_id=user.id, inputs_json=inputs)`
- Commit
- Dispatch: `from app.tasks.agent_tasks import run_agent_job; run_agent_job.delay(job.id)`
- Return HTMX partial: `<div id="agent-run-result" hx-get="/ui/agents/jobs/{job.id}/status" hx-trigger="load, every 2s" hx-swap="outerHTML" aria-live="polite">` with spinner + "Агент работает..."

**GET /ui/agents/jobs/{job_id}/status** — HTMX poll
- Load job via `get_job(db, job_id)`
- Render `_agent_job_status.html` partial with `job=job`
- Template handles 3 states: pending/running (keep polling), done (no hx-trigger), error (no hx-trigger + retry button)

**POST /ui/agents/{agent_id}/favourite** — HTMX toggle
- Call `toggle_favourite(db, user_id=user.id, agent_id=agent_id)`
- Commit
- Return star icon HTML with `hx-post="/ui/agents/{agent_id}/favourite" hx-swap="outerHTML" hx-target="this"`
- Filled star (`text-amber-400`) if now favourited, outline (`text-gray-300`) if removed
- Include `aria-label` per UI-SPEC accessibility notes

**POST /ui/agents/{agent_id}/fork** — fork agent
- Call `fork_agent(db, original_id=agent_id, created_by=user.id)`
- Commit
- Redirect to `/ui/agents/{new_agent.id}/edit` with 302
