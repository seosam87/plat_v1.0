Create `app/services/audit_service.py`:

### Pure detection functions (no DB, testable)

**1. `detect_author_block(html: str) -> bool`**
Parse rendered HTML for author block patterns:
- CSS selectors as regex: `.author-box`, `.post-author`, `.author-info`, `.entry-author`, `[rel="author"]`, `.vcard.author`
- Pattern: `re.search(r'class="[^"]*\bauthor[-_]?(box|info|block|bio|meta)\b', html, re.I)`
- Additional: check for `rel="author"` attribute
- Returns True if any pattern matched

**2. `detect_related_posts(html: str) -> bool`**
Parse rendered HTML for related posts block:
- Patterns: `.related-posts`, `.yarpp-related`, `.jp-relatedposts`, `.related_posts`, `.crp_related`, `.similar-posts`
- Pattern: `re.search(r'class="[^"]*\b(related[-_]?posts?|yarpp|similar[-_]?posts?|crp_related|jp-relatedposts)\b', html, re.I)`
- Additional: check for text "Похожие статьи", "Читайте также", "Related Posts", "Вам также может понравиться"
- Returns True if any pattern matched

**3. `detect_cta_block(html: str) -> bool`**
Check for CTA-like blocks:
- Patterns: `.cta-block`, `.cta-section`, `.call-to-action`, `[class*="cta"]`
- Pattern: `re.search(r'class="[^"]*\bcta[-_]?(block|section|banner|widget)?\b', html, re.I)`
- Additional: look for common CTA button texts: "Заказать", "Оставить заявку", "Купить", "Получить консультацию"
- Returns True if any pattern matched

**4. `classify_content_type(page_type: str, url: str) -> str`**
Determine content_type from page_type and URL:
- `product` → `commercial`
- `article` → `informational`
- `category` → `commercial`
- `landing` → check URL patterns: if contains `/uslugi/`, `/services/`, `/price/`, `/catalog/` → `commercial`, else `informational`
- `unknown` → `unknown`
- Returns ContentType value string

**5. `check_internal_links(internal_link_count: int, min_links: int = 1) -> bool`**
Returns True if `internal_link_count >= min_links`.

### Check engine

**6. `run_checks_for_page(html: str, page_data: dict, check_definitions: list[dict]) -> list[dict]`**
Pure function. Takes rendered HTML, page metadata dict, and list of check definitions.

`page_data` keys: `has_toc`, `has_schema`, `has_noindex`, `internal_link_count`, `content_type`, `page_type`, `url`

For each check definition:
- Skip if `is_active` is False
- Skip if `applies_to` != `unknown` and `applies_to` != `page_data["content_type"]`
- Run the appropriate check:
  - `toc_present` → `page_data["has_toc"]`
  - `schema_present` → `page_data["has_schema"]`
  - `author_block` → `detect_author_block(html)`
  - `related_posts` → `detect_related_posts(html)`
  - `cta_present` → `detect_cta_block(html)`
  - `internal_links` → `check_internal_links(page_data["internal_link_count"])`
  - `noindex_check` → `not page_data["has_noindex"]` (pass if NOT noindex)
- Return list of `{"check_code": str, "status": "pass"|"fail"|"warning", "details": str|None}`
- Status is `fail` if severity=`error` and check fails, `warning` if severity=`warning` and check fails, `pass` if check passes

### Async DB functions

**7. `async def get_active_checks(db: AsyncSession) -> list[AuditCheckDefinition]`**
Fetch all active check definitions ordered by sort_order.

**8. `async def save_audit_results(db: AsyncSession, site_id: UUID, page_url: str, results: list[dict]) -> None`**
Upsert audit results for a page. Use PostgreSQL `INSERT ... ON CONFLICT (site_id, page_url, check_code) DO UPDATE SET status=, details=, updated_at=`.

**9. `async def get_audit_results_for_site(db: AsyncSession, site_id: UUID, page_urls: list[str] | None = None) -> list[AuditResult]`**
Fetch audit results, optionally filtered by URL list. Order by page_url, check_code.

**10. `async def update_content_type(db: AsyncSession, page_id: UUID, content_type: str) -> None`**
Update a page's content_type field.

**11. `async def classify_and_update_pages(db: AsyncSession, site_id: UUID, crawl_job_id: UUID) -> int`**
For all pages of a crawl job: run `classify_content_type()` and update the `content_type` field. Returns count of pages classified.

**12. `async def get_check_definitions(db: AsyncSession) -> list[dict]`**
Fetch all check definitions as list of dicts for the UI.

**13. `async def update_check_definition(db: AsyncSession, check_id: UUID, **fields) -> AuditCheckDefinition | None`**
Update a check definition (name, description, is_active, severity, sort_order). Returns updated object.

**14. `async def create_check_definition(db: AsyncSession, code: str, name: str, applies_to: str = "unknown", severity: str = "warning", auto_fixable: bool = False, fix_action: str | None = None) -> AuditCheckDefinition`**
Create a new user-defined check. Returns created object.
