<action>
Create `app/models/audit.py` with three SQLAlchemy models:

**AuditCheckDefinition** — configurable checklist items:
- `id`: UUID PK, default uuid4
- `code`: String(50), unique, not null — machine-readable check identifier (e.g. `toc_present`, `schema_present`, `cta_present`, `author_block`, `related_posts`)
- `name`: String(255), not null — display name (Russian: "Наличие TOC", "Schema.org разметка")
- `description`: Text, nullable — what the check does
- `applies_to`: SAEnum(ContentType), not null, default `unknown` — which content_type this check applies to. `unknown` means applies to all.
- `is_active`: Boolean, default True — user can disable checks
- `severity`: String(20), not null, default `warning` — `error` or `warning`
- `auto_fixable`: Boolean, default False — whether the system can fix this automatically
- `fix_action`: String(50), nullable — action code for auto-fix (e.g. `inject_toc`, `inject_schema`, `inject_cta`)
- `sort_order`: Integer, default 0 — display order in checklist UI
- `created_at`: DateTime(timezone=True), default utcnow

**AuditResult** — per-page check results:
- `id`: UUID PK, default uuid4
- `site_id`: UUID FK → sites.id CASCADE, not null
- `page_url`: String(2000), not null
- `check_code`: String(50), not null — references AuditCheckDefinition.code
- `status`: String(20), not null — `pass`, `fail`, `warning`, `fixed`, `skipped`
- `details`: Text, nullable — extra info (e.g. "TOC added on 2026-04-02")
- `wp_content_job_id`: UUID FK → wp_content_jobs.id SET NULL, nullable — link to pipeline job if auto-fixed
- `created_at`: DateTime(timezone=True), default utcnow
- `updated_at`: DateTime(timezone=True), default utcnow, onupdate utcnow
- `__table_args__`: UniqueConstraint('site_id', 'page_url', 'check_code', name='uq_audit_result_site_page_check')

**SchemaTemplate** — user-customizable schema.org templates:
- `id`: UUID PK, default uuid4
- `site_id`: UUID FK → sites.id CASCADE, nullable — null means system default (shared)
- `schema_type`: String(50), not null — e.g. `Article`, `Service`, `Product`, `LocalBusiness`, `FAQ`
- `name`: String(255), not null — display name
- `template_json`: Text, not null — JSON-LD template with placeholders like `{{title}}`, `{{url}}`, `{{description}}`
- `is_default`: Boolean, default False — system-provided default template
- `created_at`: DateTime(timezone=True), default utcnow
- `__table_args__`: UniqueConstraint('site_id', 'schema_type', name='uq_schema_template_site_type')

Import pattern follows existing models:
```python

from sqlalchemy import Boolean, DateTime, ForeignKey, Integer, String, Text, UniqueConstraint

from app.database import Base

```

Table names: `audit_check_definitions`, `audit_results`, `schema_templates`.
