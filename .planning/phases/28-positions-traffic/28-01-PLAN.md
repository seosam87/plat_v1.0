**1. Create `app/services/mobile_positions_service.py`** (new file, ~60 lines).

Follow `mobile_digest_service.py` pattern: async-only, `AsyncSession` parameter, `from loguru import logger`.

Function `get_mobile_positions(db: AsyncSession, site_id: uuid.UUID, period_days: int | None = None, dropped_only: bool = False, limit: int = 100) -> list[dict]`:
- Query `keyword_latest_positions` JOIN `keywords` to get `phrase`
- SELECT: `KeywordLatestPosition.id`, `.position`, `.previous_position`, `.delta`, `.engine`, `.checked_at`, `Keyword.phrase`
- JOIN: `Keyword, KeywordLatestPosition.keyword_id == Keyword.id`
- WHERE: `KeywordLatestPosition.site_id == site_id`
- If `dropped_only=True`: add `.where(KeywordLatestPosition.delta < 0)`
- If `period_days` not None: add `.where(KeywordLatestPosition.checked_at >= datetime.utcnow() - timedelta(days=period_days))`
- ORDER BY: `func.abs(KeywordLatestPosition.delta).desc().nullslast()`
- LIMIT: `limit`
- Return list of dicts with keys: `id`, `phrase`, `position`, `previous_position`, `delta`, `engine`, `checked_at`
