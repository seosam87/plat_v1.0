**app/services/paa_service.py** — PAA extraction from Yandex SERP HTML:

`extract_paa_blocks(html: str) -> list[dict]`:
- Uses BeautifulSoup4 with lxml parser per D-07
- Returns list of `{"question": str, "source_block": str}` where source_block is "частые вопросы" or "похожие запросы"
- Per D-08: first level only, no recursive expansion
- Per D-10: extract BOTH block types

Strategy (per research Pitfall 3 — use text-content matching, not class selectors):
1. For "Частые вопросы": find headings (h2, h3, div) containing text "частые вопросы" (case-insensitive). Walk parent container, extract all li/a/span children with text length > 10 chars.
2. For "Похожие запросы": find headings containing "похожие запросы" (case-insensitive). Same extraction.
3. Fallback: also search for `[data-fast-name]` attributes containing "question" or "related"
4. Deduplicate questions (same text can appear in multiple selectors)
5. Log warning via loguru if 0 results found for a phrase (Pitfall 3 warning sign)

`extract_paa_for_phrase(phrase: str, xmlproxy_html: str) -> list[dict]`:
- Wrapper that calls `extract_paa_blocks(xmlproxy_html)`
- Prepends `{"phrase": phrase}` to each result dict
- Returns list of `{"phrase": str, "question": str, "source_block": str}`

**app/tasks/paa_tasks.py** — Single Celery task:

`run_paa(self, job_id: str)`:
- `@celery_app.task(name="app.tasks.paa_tasks.run_paa", bind=True, max_retries=3, soft_time_limit=300, time_limit=360)`
- Follow `run_commerce_check` pattern exactly
- Load PAAJob by UUID from `get_sync_db()`, set status="running", commit
- For each phrase in input_phrases:
  - Call XMLProxy to get Yandex SERP HTML (need to get HTML body from XMLProxy, not just XML positions — check xmlproxy_service for a method that returns raw HTML. If none exists, add `fetch_yandex_html_sync(query, region_id) -> str` that fetches SERP page HTML via XMLProxy)
  - Call `extract_paa_for_phrase(phrase, html)`
  - Create PAAResult rows for each extracted question
  - `time.sleep(0.5)` between phrases (XMLProxy rate limiting)
- On XMLProxyError: `self.retry(exc=exc)`
- On completion: set job.status="complete", job.result_count=total_results, job.completed_at=now
- On failure: set job.status="failed", job.error_message=str(exc)
