# Plan v3-09-01: Intent Auto-Detect — full phase

<objective>
Create proxy-enabled SERP parser (Playwright + proxy rotation + anticaptcha), intent detection service that analyzes SERP TOP-10 for commercial vs informational dominance, batch Celery task for processing unclustered keywords, semi-auto UI for review and confirm.
</objective>

<tasks>

<task id="01">
<title>Add proxy/anticaptcha config and create proxy_serp_service.py</title>
<read_first>
- app/services/serp_parser_service.py
- app/config.py
</read_first>
<action>
Add to `app/config.py`:
```python
PROXY_URL: str = ""  # e.g. http://user:pass@proxy:port
ANTICAPTCHA_KEY: str = ""
SERP_DAILY_LIMIT: int = 200  # raised from 50 with proxy
```

Create `app/services/proxy_serp_service.py`:

**`async def parse_serp_with_proxy(query: str, engine: str = "yandex", region: str = "") -> dict`**
1. Launch Playwright browser context with proxy: `browser.new_context(proxy={"server": PROXY_URL})`
2. Navigate to Yandex/Google SERP
3. If CAPTCHA detected → solve via anticaptcha API (POST to `https://api.anti-captcha.com/createTask`)
4. Extract TOP-10: [{position, url, domain, title, snippet}]
5. Detect SERP features
6. Return `{results: [...], features: [...], engine, query}`

**`async def solve_captcha(page, anticaptcha_key: str) -> bool`**
Send page screenshot/sitekey to anticaptcha, wait for solution, inject.

**`def is_proxy_configured() -> bool`**
Check PROXY_URL and ANTICAPTCHA_KEY are set.

Fallback: if proxy not configured, use existing `serp_parser_service.parse_serp()`.
</action>

<task id="02">
<title>Create intent_service.py</title>
<read_first>
- app/services/serp_analysis_service.py (classify_site_type)
- app/models/cluster.py (KeywordCluster.intent)
- app/models/analytics.py (SessionSerpResult — cached SERP)
</read_first>
<action>
Create `app/services/intent_service.py`:

**`def detect_intent_from_serp(results: list[dict]) -> dict`** (pure)
Analyze TOP-10 results:
- Count commercial vs informational sites (using `classify_site_type`)
- If commercial >= 7 → `commercial`
- If informational >= 7 → `informational`
- If mixed → `mixed` (requires manual decision)
- Return `{intent: str, commercial_count: int, informational_count: int, confidence: float}`

Confidence = max(commercial, informational) / total.

**`IntentProposal`** — not a model, just a dict: `{keyword_id, phrase, proposed_intent, confidence, serp_summary}`

**Async functions:**
- `async def batch_detect_intent(db, site_id, keyword_ids, use_cache=True) -> list[dict]`
  1. For each keyword, check SessionSerpResult cache first
  2. If not cached, parse SERP via proxy_serp_service
  3. Run detect_intent_from_serp
  4. Return list of IntentProposal dicts

- `async def confirm_intent(db, keyword_id, intent) -> None`
  Update keyword's cluster intent.

- `async def bulk_confirm_intents(db, proposals: list[dict]) -> int`
  Apply all proposed intents. Return count.

- `async def get_unclustered_keywords(db, site_id, limit=100) -> list[dict]`
  Keywords where cluster intent is 'unknown' or no cluster.
</action>
