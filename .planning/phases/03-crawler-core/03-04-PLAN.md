## Objective

Add crawl history and change feed API endpoints, a dedicated crawl router, Jinja2 templates for the change feed UI, and update the sites index template to show crawl history links.

## Tasks

1. Add `GET /sites/{site_id}/crawls` endpoint to `app/routers/sites.py`
2. Create `app/routers/crawl.py` with:
   - `GET /crawls/{crawl_job_id}/pages` — filtered pages list (all/seo_changed/content_changed/new_pages/status_changed)
   - `GET /ui/sites/{site_id}/crawls` — HTML list of crawl jobs for a site
   - `GET /ui/crawls/{crawl_job_id}` — HTML change feed page
3. Register crawl router in `app/main.py`
4. Create `app/templates/crawl/` with `feed.html` (change feed) and `history.html` (crawl list)
5. Update `app/templates/sites/index.html` to add Crawl History link per site
6. Create `tests/test_crawl_api.py`
7. Run tests and fix failures

## Verification

`docker compose run --rm api pytest tests/test_crawl_api.py -x -q`
