# Plan v3-04-05: Analytics router

<objective>
Create the FastAPI analytics router with endpoints for the full workflow: keyword filtering, session CRUD, position check trigger, SERP parse trigger, competitor crawl trigger, comparison data, brief generation, and export. Register in main.py.
</objective>

<tasks>

<task id="01">
<title>Create analytics router with all endpoints</title>
<read_first>
- app/routers/audit.py (router pattern)
- app/services/analytics_service.py
- app/services/serp_analysis_service.py
- app/services/brief_service.py
- app/tasks/analytics_tasks.py
</read_first>
<action>
Create `app/routers/analytics.py`:

Router: `prefix="/analytics"`, tags `["analytics"]`

**Filter & session:**
1. `GET /sites/{site_id}/keywords` — Advanced keyword filter. All filter params as query. Returns paginated keyword list + total.
2. `GET /sites/{site_id}/filter-options` — Available filter values (clusters, groups, regions, etc.).
3. `POST /sites/{site_id}/sessions` — Create session from keyword selection. Body: `{name, keyword_ids, filters_applied}`.
4. `GET /sites/{site_id}/sessions` — List sessions for site.
5. `GET /sessions/{session_id}` — Get session details + keywords.
6. `DELETE /sessions/{session_id}` — Delete session.

**Workflow triggers:**
7. `POST /sessions/{session_id}/check-positions` — Trigger group position check (Celery).
8. `POST /sessions/{session_id}/parse-serp` — Trigger SERP parsing (Celery).
9. `POST /sessions/{session_id}/crawl-competitor` — Trigger competitor crawl. Body: `{mode: "light"|"full", domain?: str}`.
10. `POST /sessions/{session_id}/set-competitor` — Manually set competitor domain.

**Results:**
11. `GET /sessions/{session_id}/serp-summary` — SERP analysis summary (top competitors, site types).
12. `GET /sessions/{session_id}/competitor-data` — Competitor page data for session.
13. `GET /sessions/{session_id}/comparison` — Our pages vs competitor side-by-side.

**Brief:**
14. `POST /sessions/{session_id}/brief` — Generate content brief. Body: `{target_url?, structure_notes?}`.
15. `GET /briefs/{brief_id}` — Get brief details.
16. `GET /briefs/{brief_id}/export` — Export brief as plain text. Query: `format=text|csv`.
17. `GET /sites/{site_id}/briefs` — List all briefs for site.
18. `DELETE /briefs/{brief_id}` — Delete brief.

**Export:**
19. `GET /sessions/{session_id}/export` — Export session keywords as CSV.

All endpoints gated behind `require_admin`.
</action>
