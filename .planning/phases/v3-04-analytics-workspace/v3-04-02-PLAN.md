Create `app/services/analytics_service.py`:

### Advanced keyword filter

**`async def filter_keywords(db, site_id, *, frequency_min=None, frequency_max=None, position_min=None, position_max=None, intent=None, cluster_id=None, group_id=None, region=None, engine=None, search=None, has_target_url=None, limit=500, offset=0) -> tuple[list[dict], int]`**

Build a SQLAlchemy query on Keyword + latest KeywordPosition (subquery for latest position per keyword):
- Join keyword with latest position via subquery
- Apply all filter conditions
- Return (list of keyword dicts with position data, total count)

Each result dict: `{id, phrase, frequency, region, engine, cluster_id, cluster_name, group_id, target_url, intent, latest_position, latest_url, delta}`

### Session CRUD

**`async def create_session(db, site_id, name, keyword_ids, filters_applied=None) -> AnalysisSession`**
**`async def get_session(db, session_id) -> AnalysisSession | None`**
**`async def list_sessions(db, site_id, limit=20) -> list[AnalysisSession]`**
**`async def update_session_status(db, session_id, status) -> AnalysisSession`**
**`async def delete_session(db, session_id) -> bool`**
**`async def get_session_keywords(db, session_id) -> list[dict]`** — fetch full keyword data for session's keyword_ids
**`async def set_session_competitor(db, session_id, domain) -> AnalysisSession`**
**`async def export_session_csv(db, session_id) -> str`** — generate CSV string with keywords + positions

### Helpers

**`async def get_filter_options(db, site_id) -> dict`** — returns available filter values:
```python
{
    "clusters": [{id, name, intent}],
    "groups": [{id, name}],
    "regions": [str],
    "engines": [str],
    "intents": [str],
    "frequency_range": {min, max},
}
```
