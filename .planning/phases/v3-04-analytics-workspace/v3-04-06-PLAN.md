# Plan v3-04-06: Analytics wizard UI

<objective>
Build the "Аналитика" page with wizard-like step-by-step interface. Steps: (1) Filter keywords → (2) Save session → (3) Check positions → (4) SERP analysis → (5) Competitor comparison → (6) Content brief. Each step loads via HTMX after the previous completes. Add nav link and site detail button.
</objective>

<tasks>

<task id="01">
<title>Create main analytics page template with wizard layout</title>
<read_first>
- app/templates/audit/index.html (UI pattern)
- app/templates/metrika/index.html (chart pattern)
</read_first>
<action>
Create `app/templates/analytics/index.html`:

Extends `base.html`. Structure:

**1. Header:** "Аналитика: {site.name}" + session selector dropdown (existing sessions) + "Новая сессия" button

**2. Step indicator bar:** 6 steps shown horizontally, current step highlighted:
- Фильтр ключей → Сессия → Позиции → SERP → Конкурент → ТЗ

**3. Step 1: Keyword filter panel** (default visible):
- All filter inputs: search, frequency min/max, position min/max, intent select, cluster select, group select, region, engine
- "Найти" button → loads filtered results via fetch()
- Results table: phrase, frequency, position, delta, target_url, cluster, group
- Checkbox per row for selection, "Выбрать все" toggle
- "Сохранить сессию" button → creates session from selected keyword_ids

**4. Step 2: Session detail** (loads after session created/selected):
- Session name, keyword count, status, created date
- Keywords table with positions
- Action buttons: "Проверить позиции" → "Анализ SERP" → depends on status
- Export CSV button

**5. Step 3-4: SERP results** (loads after SERP parse completes):
- Top competitors table: domain, appearances, avg position, site type badge
- Site type distribution: commercial/info/aggregator counts
- Auto-selected competitor highlighted, option to change
- "Краулить конкурента" button with mode selector (лёгкий/полный)

**6. Step 5: Comparison** (loads after competitor crawl):
- Side-by-side table: our page vs competitor page
- Columns: URL, Title, H1, Meta, Schema, TOC, Word count, Links
- Green/red highlights for differences

**7. Step 6: Content brief** (loads after brief generated):
- Brief preview with all sections
- "Экспорт ТЗ" button (text format download)
- "Экспорт ключей" button (CSV download)

All copy in Russian. JS functions for step navigation and async task polling.
</action>
