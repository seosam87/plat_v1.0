<read_first>

<action>
Create `app/services/serp_analysis_service.py`:

### Pure analysis functions

**`classify_site_type(domain: str, url: str) -> str`**
Heuristic classification:
- `aggregator` — if domain matches known aggregators (avito.ru, ozon.ru, wildberries.ru, yandex.ru/maps, 2gis.ru, zoon.ru, etc.)
- `informational` — if URL contains /blog/, /wiki/, /article/, /novosti/, or domain is wikipedia/habr/vc.ru
- `commercial` — default for everything else

**`analyze_serp_results(serp_data: list[dict]) -> dict`**
Takes list of SERP results across multiple keywords [{keyword_phrase, results: [{position, url, domain, title}]}].
Returns:
```python
{
    "top_competitors": [{domain, appearances, avg_position, site_type}],  # sorted by appearances desc
    "site_type_distribution": {"commercial": N, "informational": N, "aggregator": N},
    "total_keywords": N,
}
```

**`extract_domains_from_serp(results_json: list[dict]) -> list[str]`**
Extract unique domains from SERP results.

### DB functions (async)

**`async def save_serp_results(db, session_id, keyword_id, keyword_phrase, results_json, features=None)`**
Save/upsert SessionSerpResult.

**`async def get_session_serp_summary(db, session_id) -> dict`**
Load all SessionSerpResults for session, run analyze_serp_results(), return summary.

**`async def get_top_competitor(db, session_id) -> str | None`**
Get the domain that appears most in TOP-10 across session keywords (excluding our own site).

<action>
Create `app/tasks/analytics_tasks.py`:

**Task 1: `check_group_positions`**
```python
@celery_app.task(bind=True, max_retries=2, queue="default", soft_time_limit=180, time_limit=210)
def check_group_positions(self, session_id: str) -> dict:
```
Steps:
1. Load session, get keyword_ids
2. For each keyword, check position using existing DataForSEO/GSC/Playwright flow
3. Save positions to keyword_positions table
4. Update session status to `positions_checked`
5. Return {checked: N}

**Task 2: `parse_group_serp`**
```python
@celery_app.task(bind=True, max_retries=2, queue="default", soft_time_limit=300, time_limit=360)
def parse_group_serp(self, session_id: str) -> dict:
```
Steps:
1. Load session keyword_ids + phrases
2. For each keyword (up to 50), parse SERP via Playwright or DataForSEO
3. Save each result as SessionSerpResult
4. Run analyze_serp_results() to find top competitors
5. Auto-set session.competitor_domain to top competitor
6. Update session status to `serp_parsed`
7. Return {parsed: N, top_competitor: str}

**Task 3: `crawl_competitor_pages`**
```python
@celery_app.task(bind=True, max_retries=1, queue="crawl", soft_time_limit=180, time_limit=210)
def crawl_competitor_pages(self, session_id: str, mode: str = "light") -> dict:
```
Steps:
1. Load session + SERP results
2. Find competitor URLs from SERP (filter by session.competitor_domain)
3. For each URL, Playwright-crawl: extract title, H1, meta, word_count, schema, toc, headings, internal_links
4. If mode="light": skip word_count and internal_links
5. Save as CompetitorPageData
6. Update session status to `compared`
7. Return {pages_crawled: N}

Register all in `celery_app.py` include list.
</action>
