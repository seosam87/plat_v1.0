# Plan v3-04-04: Content brief service

<objective>
Create the brief generation service: build a content brief (ТЗ) from an analysis session. Includes recommended SEO fields, keywords with frequencies, heading structure derived from competitors and cluster data, site architecture placement, and export to text/CSV.
</objective>

<action>
Create `app/services/brief_service.py`:

### Pure functions

**`build_heading_structure(competitor_headings: list[list[dict]], keywords: list[str]) -> list[dict]`**
Analyze competitor headings (from CompetitorPageData.headings_json) across multiple competitor pages.
- Find common H2 patterns (appearing on 2+ competitors)
- Suggest H3 sub-headings based on keyword phrases
- Return `[{level: 2|3, text: str, source: "competitor"|"keyword"}]`

**`suggest_seo_fields(keyword_phrase: str, site_name: str, competitor_titles: list[str]) -> dict`**
Generate recommended SEO fields:
- `title`: keyword_phrase + site_name (max 60 chars)
- `h1`: keyword_phrase (max 70 chars)
- `meta_description`: template with keyword + call to action (max 160 chars)

**`format_brief_text(brief: dict) -> str`**
Format a ContentBrief dict as readable plain text for export:
```
ТЗ: {title}
URL: {target_url}
Дата: {created_at}

== SEO-поля ==
Title: {recommended_title}
H1: {recommended_h1}
Meta Description: {recommended_meta}

== Ключевые слова ({count}) ==
{phrase} — {frequency}
...

== Структура заголовков ==
H2: {text}
  H3: {text}
...

== Место в структуре ==
{structure_notes}

== Данные конкурентов ==
{competitor summary}
```

### Async DB functions

**`async def generate_brief(db, session_id, target_url=None, structure_notes=None) -> ContentBrief`**
1. Load session + keywords + competitor page data
2. Run `build_heading_structure()` from competitor headings
3. Run `suggest_seo_fields()` from top keyword
4. Create ContentBrief record
5. Update session status to `brief_created`
6. Return brief

**`async def get_brief(db, brief_id) -> ContentBrief | None`**
**`async def list_briefs(db, site_id) -> list[ContentBrief]`**
**`async def delete_brief(db, brief_id) -> bool`**
**`async def export_brief_text(db, brief_id) -> str`** — get brief, format as text
**`async def export_brief_csv(db, brief_id) -> str`** — keywords section as CSV
