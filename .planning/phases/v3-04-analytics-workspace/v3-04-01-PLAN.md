<action>
Create `app/models/analytics.py`:

**SessionStatus** enum: `draft`, `positions_checked`, `serp_parsed`, `competitor_found`, `compared`, `brief_created`, `completed`

**AnalysisSession**:
- `id`: UUID PK
- `site_id`: UUID FK тЖТ sites.id CASCADE
- `name`: String(255), not null тАФ user-given name
- `status`: SAEnum(SessionStatus), default `draft`
- `keyword_ids`: JSON, not null тАФ list of UUID strings
- `filters_applied`: JSON, nullable тАФ {frequency_min, frequency_max, position_min, position_max, intent, cluster_id, region, engine} тАФ for reference
- `keyword_count`: Integer, default 0
- `competitor_domain`: String(500), nullable тАФ detected/selected top competitor
- `notes`: Text, nullable
- `created_at`: DateTime(timezone=True)
- `updated_at`: DateTime(timezone=True), onupdate

**SessionSerpResult** тАФ SERP TOP-10 per keyword per session:
- `id`: UUID PK
- `session_id`: UUID FK тЖТ analysis_sessions.id CASCADE
- `keyword_id`: UUID, not null
- `keyword_phrase`: String(1000), not null тАФ denormalized for display
- `results_json`: JSON, not null тАФ list of {position, url, domain, title}
- `features`: JSON, nullable тАФ list of SERP features detected
- `parsed_at`: DateTime(timezone=True)
- `__table_args__`: UniqueConstraint('session_id', 'keyword_id', name='uq_serp_result_session_keyword')

**CompetitorPageData** тАФ one-time crawl of competitor page:
- `id`: UUID PK
- `session_id`: UUID FK тЖТ analysis_sessions.id CASCADE
- `url`: String(2000), not null
- `domain`: String(500), not null
- `title`: Text, nullable
- `h1`: Text, nullable
- `meta_description`: Text, nullable
- `word_count`: Integer, nullable
- `has_schema`: Boolean, default False
- `has_toc`: Boolean, default False
- `internal_link_count`: Integer, nullable
- `headings_json`: JSON, nullable тАФ [{level, text}]
- `crawl_mode`: String(20), default 'light' тАФ 'light' or 'full'
- `crawled_at`: DateTime(timezone=True)

**ContentBrief** тАФ generated ╨в╨Ч:
- `id`: UUID PK
- `session_id`: UUID FK тЖТ analysis_sessions.id CASCADE
- `site_id`: UUID FK тЖТ sites.id CASCADE
- `title`: String(255), not null тАФ brief name
- `target_url`: String(2000), nullable тАФ recommended URL
- `recommended_title`: Text, nullable
- `recommended_h1`: Text, nullable
- `recommended_meta`: Text, nullable
- `keywords_json`: JSON, not null тАФ [{phrase, frequency}]
- `headings_json`: JSON, nullable тАФ [{level, text}] recommended H2/H3
- `structure_notes`: Text, nullable тАФ where this page fits in site architecture
- `competitor_data_json`: JSON, nullable тАФ summary of competitor analysis
- `created_at`: DateTime(timezone=True)

Table names: `analysis_sessions`, `session_serp_results`, `competitor_page_data`, `content_briefs`.
</action>
