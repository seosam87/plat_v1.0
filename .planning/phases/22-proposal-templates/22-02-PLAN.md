Endpoints (per D-11: admin-only for writes, all-authenticated for reads/preview):

2. **GET `/ui/templates/new`** — `template_new_page` — `require_admin`
   - Fetch all clients: `select(Client).where(Client.is_deleted == False).order_by(Client.company_name)`
   - Render `proposal_templates/edit.html` with `{"template": None, "clients": clients, "sites": [], "current_user": current_user}`

3. **GET `/ui/templates/{template_id}/edit`** — `template_edit_page` — `require_admin`
   - Fetch template via `template_service.get_template(db, template_id)`, 404 if None
   - Fetch all clients (same query as new)
   - Render `proposal_templates/edit.html` with `{"template": template, "clients": clients, "sites": [], "current_user": current_user}`

4. **POST `/ui/templates`** — `template_create` — `require_admin`
   - Accept Form fields: `name: str`, `template_type: str`, `description: str = ""`, `body: str = ""`
   - Validate name is non-empty; convert template_type string to TemplateType enum
   - Call `template_service.create_template(db, name, template_type_enum, description or None, body, current_user.id)`
   - Return redirect response: `RedirectResponse(url=f"/ui/templates/{new.id}/edit", status_code=303)`

5. **POST `/ui/templates/{template_id}`** — `template_update` — `require_admin`
   - Accept Form fields: `name: str`, `template_type: str`, `description: str = ""`, `body: str = ""`
   - Call `template_service.update_template(db, template_id, name, template_type_enum, description or None, body)`, 404 if None
   - Return HTMLResponse with HX-Trigger header: `{"showToast": "Шаблон сохранён"}`
   - Response body: small HTML snippet for `#save-status` swap (e.g., `<span>Сохранено</span>`)

6. **DELETE `/ui/templates/{template_id}`** — `template_delete` — `require_admin`
   - hx-confirm is client-side (per UI-SPEC delete confirmation pattern)
   - Call `template_service.delete_template(db, template_id)`, 404 if False
   - Return empty HTMLResponse with HX-Trigger: `{"showToast": "Шаблон удалён"}`
   - Set HX-Redirect header to `/ui/templates` to reload list

7. **POST `/ui/templates/{template_id}/clone`** — `template_clone` — `require_admin`
   - Call `template_service.clone_template(db, template_id, current_user.id)`, 404 if None
   - Per D-12: return HTMLResponse with HX-Redirect: `/ui/templates/{clone.id}/edit`
   - Set HX-Trigger: `{"showToast": "Шаблон скопирован — откройте его для редактирования"}`

8. **POST `/ui/templates/preview`** — `template_preview` — `require_any_authenticated`
   - Accept Form fields: `body: str`, `client_id: str`, `site_id: str`
   - Parse client_id and site_id as UUIDs; return error HTML if invalid
   - Call `resolve_template_variables(db, client_id_uuid, site_id_uuid)`
   - Call `render_template_preview(body, context)`
   - Return `HTMLResponse(content=rendered_html)` — raw HTML, NO base.html wrapper (per D-14 iframe isolation, per Pitfall 3)

9. **GET `/ui/templates/sites`** — `sites_for_client` — `require_any_authenticated`
   - Accept Query param: `client_id: uuid.UUID`
   - Query `select(Site).where(Site.client_id == client_id, Site.is_active == True).order_by(Site.url)`
   - Return HTMLResponse with `<select id="site-select" name="site_id">` containing `<option>` elements for each site (url as display text, id as value)
   - This is the HTMX dependent select target (per D-13, same pattern as CRM)

**IMPORTANT routing order:** The `/sites` and `/preview` routes (paths without `{template_id}`) MUST be defined BEFORE `/{template_id}/edit` and `/{template_id}/clone` routes in the router, otherwise FastAPI will try to parse "sites" or "preview" as a UUID and return 422.

```javascript
import { basicSetup } from 'https://esm.sh/@codemirror/basic-setup@0.20';
import { EditorView } from 'https://esm.sh/@codemirror/view@6';
import { html } from 'https://esm.sh/@codemirror/lang-html@6';

const initialContent = document.getElementById('template-body-initial')?.textContent || '';
const view = new EditorView({
  doc: initialContent,
  extensions: [basicSetup, html()],
  parent: document.getElementById('codemirror-mount'),
});

// Variable insert function (per D-06, UI-SPEC interaction contract)
window.insertVariable = function(varName) {
  const pos = view.state.selection.main.head;
  const insert = '{{ ' + varName + ' }}';
  view.dispatch({
    changes: { from: pos, insert: insert },
    selection: { anchor: pos + insert.length },
  });
  view.focus();
};

// Sync editor content to hidden input before form submit (per Pitfall 2)
document.getElementById('save-form')?.addEventListener('submit', function() {
  document.getElementById('body-hidden').value = view.state.doc.toString();
});

// Preview button: fetch rendered HTML and set iframe srcdoc (per D-14, Pitfall 3)
document.getElementById('preview-btn')?.addEventListener('click', async function() {
  const body = view.state.doc.toString();
  const clientId = document.getElementById('client-select')?.value;
  const siteId = document.getElementById('site-select')?.value;
  if (!clientId || !siteId) return;
  
  const iframe = document.getElementById('preview-iframe');
  iframe.srcdoc = '<p style="color:#6b7280;padding:24px;">Загрузка превью...</p>';
  
  try {
    const resp = await fetch('/ui/templates/preview', {
      method: 'POST',
      headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
      body: new URLSearchParams({ body, client_id: clientId, site_id: siteId }),
    });
    const html = await resp.text();
    iframe.srcdoc = html;
  } catch (err) {
    iframe.srcdoc = '<p style="color:#dc2626;padding:24px;">Ошибка рендеринга превью. Проверьте синтаксис Jinja2 в шаблоне.</p>';
  }
});
```
