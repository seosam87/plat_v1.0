```python
class TemplateType(str, PyEnum):

    id: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), primary_key=True, default=uuid.uuid4)
    name: Mapped[str] = mapped_column(String(255), nullable=False)

    description: Mapped[str | None] = mapped_column(Text, nullable=True)

    updated_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), default=datetime.utcnow, onupdate=datetime.utcnow, nullable=False)
```

5. `async def delete_template(db, template_id: uuid.UUID) -> bool` — fetch, `await db.delete(obj)`, commit, return True; return False if not found
6. `async def clone_template(db, template_id: uuid.UUID, created_by_id: uuid.UUID) -> ProposalTemplate | None` — per D-12: fetch original, create new with `name=f"{original.name} (копия)"`, same template_type/description/body, new UUID, commit, return clone

Create `app/services/template_variable_resolver.py` with:

1. `class _HighlightUndefined(Undefined)` — `__str__` returns `<span class="unresolved-var" style="background:#fef3c7;color:#92400e;padding:0 4px;border-radius:2px;font-family:monospace;font-size:0.85em;">{{ {self._undefined_name} }}</span>` (per D-07)
2. `async def resolve_template_variables(db: AsyncSession, client_id: uuid.UUID, site_id: uuid.UUID) -> dict` — queries:
   - Client row: `select(Client).where(Client.id == client_id, Client.is_deleted == False)`
   - Site row: `select(Site).where(Site.id == site_id)`
   - GSC count: `select(func.count()).select_from(OAuthToken).where(OAuthToken.site_id == site_id, OAuthToken.provider == "gsc")`
   - Last crawl: `text("SELECT MAX(created_at) FROM crawl_jobs WHERE site_id = :sid AND status = 'done'")`
   - Audit errors: `text("SELECT COUNT(DISTINCT page_url) FROM error_impact_scores WHERE site_id = :sid")`
   - Top-10 positions: CTE with `DISTINCT ON (keyword_id, engine)` then `COUNT(*) FILTER (WHERE position <= 10)` from keyword_positions
   - Manager name: second query on User table via client.manager_id
   - Returns flat dict: `{"client": {"name": ..., "legal_name": ..., "inn": ..., "email": ..., "phone": ..., "manager": ...}, "site": {"url": ..., "domain": urlparse(url).netloc, "top_positions_count": int, "audit_errors_count": int, "last_crawl_date": str, "gsc_connected": bool, "metrika_id": str}}`
   - CRITICAL: Never pass ORM model objects — only plain str/int/bool values in the dict (per Pitfall 1 from RESEARCH.md)
3. `def render_template_preview(body: str, context: dict) -> str` — `SandboxedEnvironment(undefined=_HighlightUndefined)`, `env.from_string(body).render(context)`, wrap in try/except returning error HTML on failure
