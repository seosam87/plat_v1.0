## What Was Built

### Task 1: XMLProxy Position Pipeline

Updated `app/tasks/position_tasks.py` with:

- **Engine split** in `check_positions()`: keywords are separated into `yandex_kws` (engine="yandex") and `google_kws` (engine="google" or None)
- **`_check_via_xmlproxy()`** helper function that:
  - Loads XMLProxy credentials from DB via `get_credential_sync(db, "xmlproxy")`
  - Fetches balance **once** before the keyword loop via `fetch_balance_sync()`
  - Guards against zero balance: sends Telegram alert and returns 0
  - Warns on low balance (`< XMLPROXY_LOW_BALANCE_THRESHOLD`, default 50 RUB)
  - Calls `search_yandex_sync(user, key, kw.phrase, lr=site.yandex_region or 213)` per keyword
  - Handles error codes: -55 (retry), -32 (stop+alert), -34 (stop+alert), -132 (skip keyword)
  - Writes positions via `write_position_sync()` with `engine_str="yandex"`
- **Google keywords** logged as "Google keyword skipped: no source configured" when DataForSEO is not configured
- **`XMLPROXY_LOW_BALANCE_THRESHOLD: int = 50`** added to `app/config.py`

### Task 2: rucaptcha.com URL Replacement

In `app/services/proxy_serp_service.py`:
- `https://api.anti-captcha.com/createTask` → `https://api.rucaptcha.com/createTask`
- `https://api.anti-captcha.com/getTaskResult` → `https://api.rucaptcha.com/getTaskResult`
- Module docstring updated: "anticaptcha" → "rucaptcha"
