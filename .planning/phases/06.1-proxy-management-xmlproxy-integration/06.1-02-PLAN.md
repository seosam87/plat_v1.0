    1. Create `app/routers/proxy_admin.py` with `router = APIRouter(prefix="/admin/proxies", tags=["proxy-admin"])`:

       **Proxy CRUD endpoints** (use relative paths within router — prefix handles `/admin/proxies`):
       - `@router.post("")` — Form params: `url: str`, `proxy_type: str = "http"`. Creates Proxy in DB. Returns HTML partial (proxy_row.html) for HTMX swap. Resolves to POST /admin/proxies.
       - `@router.delete("/{proxy_id}")` — Deletes proxy by UUID. Returns empty string with 200 for HTMX outerHTML swap. Resolves to DELETE /admin/proxies/{proxy_id}.
       - `@router.get("")` — Returns all proxies as JSON list (for HTMX refresh of table body). Resolves to GET /admin/proxies.

       **Health check endpoints:**
       - `@router.post("/check-all")` — Iterates all proxies, calls `check_proxy_sync(proxy.url)` for each, updates `proxy.status`, `proxy.response_time_ms`, `proxy.last_checked_at` in DB. Also fetches XMLProxy balance (if credentials saved) and rucaptcha balance. Returns full proxy tab HTML partial for HTMX swap. Resolves to POST /admin/proxies/check-all.
       - `@router.post("/{proxy_id}/check")` — Check single proxy health, update DB, return updated proxy row partial. Resolves to POST /admin/proxies/{proxy_id}/check.

       **Credential endpoints** (IMPORTANT: use sub-paths under the /admin/proxies prefix — NOT absolute /admin/credentials paths):
       - `@router.post("/credentials/xmlproxy")` — Form params: `user: str`, `key: str`. Calls `save_credential_sync(db, "xmlproxy", {"user": user, "key": key})`. Redirects back to settings page. Resolves to POST /admin/proxies/credentials/xmlproxy.
       - `@router.post("/credentials/rucaptcha")` — Form param: `key: str`. Calls `save_credential_sync(db, "rucaptcha", {"key": key})`. Redirects back to settings page. Resolves to POST /admin/proxies/credentials/rucaptcha.
       - `@router.post("/credentials/anticaptcha")` — Form param: `key: str`. Calls `save_credential_sync(db, "anticaptcha", {"key": key})`. Redirects back to settings page. Resolves to POST /admin/proxies/credentials/anticaptcha.

       **Balance endpoints:**
       - `@router.get("/xmlproxy-balance")` — Loads XMLProxy credentials from DB via `get_credential_sync`, calls `fetch_balance_sync`, returns JSON `{"balance": float, "cur_cost": float, "max_cost": float}` or `{"error": "not configured"}`. Resolves to GET /admin/proxies/xmlproxy-balance.
       - `@router.get("/rucaptcha-balance")` — Loads rucaptcha key from DB, POSTs to `https://api.rucaptcha.com/getBalance` with `{"clientKey": key}`, returns JSON `{"balance": float}` or `{"error": "..."}`. Resolves to GET /admin/proxies/rucaptcha-balance.

       All endpoints use `get_sync_db()` for DB access (consistent with admin patterns). Import `Depends` for auth if admin routes use it (check existing admin.py pattern).

       - Test GET /admin/proxies/xmlproxy-balance returns balance or error
