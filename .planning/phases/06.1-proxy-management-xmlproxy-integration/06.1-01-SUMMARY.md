## Decisions Made

- Used `native_enum=False` for both Proxy enums so the DB stores plain VARCHAR values — avoids PG enum DDL that requires ALTER TYPE for future values
- ENCRYPTED_FIELDS dict pattern allows future services (rucaptcha, gsc, etc.) to register sensitive field names without modifying core service logic
- Credential service tests use SQLite in-memory to run without a live PG container
