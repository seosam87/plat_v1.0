       - Constants: `XMLPROXY_SEARCH = "https://xmlproxy.ru/search.php"`, `XMLPROXY_BALANCE = "https://xmlproxy.ru/balance.php"`
       - `class XMLProxyError(Exception)`: `__init__(self, code: int, message: str)`, store `self.code` and `self.message`
       - `def search_yandex_sync(user: str, key: str, query: str, lr: int = 213) -> dict`:
         - Build params: `{"user": user, "key": key, "query": query, "lr": str(lr), "groupby": "attr=d.mode=deep.groups-on-page=10", "ydomain": "ru"}`
         - Use `httpx.Client(timeout=30)` to GET `XMLPROXY_SEARCH` with params
         - Call `_parse_yandex_xml(resp.text)` and return result
       - `def fetch_balance_sync(user: str, key: str) -> dict | None`:
         - GET `XMLPROXY_BALANCE` with `{"user": user, "key": key}`, timeout=10
         - Return `resp.json()` or None on any exception
         - Log warning on failure with loguru
       - `def _parse_yandex_xml(xml_text: str) -> dict`:
         - Use `xml.etree.ElementTree.fromstring(xml_text)`
         - Check for `<error code="N">` element; if found, raise `XMLProxyError(int(code), text)`
         - Iterate `root.findall(".//group")`, extract `doc/url`, `doc/title`, `doc/domain` from each
         - If domain is empty, extract from URL (split on "/", take index 2)
         - Return `{"results": [{"position": i, "url": url, "title": title, "domain": domain}], "error_code": None}`

    2. Create `app/services/proxy_health_service.py`:
       - `HEALTH_CHECK_URL = "https://ya.ru/robots.txt"`
       - `def check_proxy_sync(proxy_url: str) -> tuple[str, int | None]`:
         - Record start time with `time.monotonic()`
         - Try `httpx.Client(proxy=proxy_url, timeout=10).get(HEALTH_CHECK_URL)`, raise_for_status()
         - On success: return `("active", int((time.monotonic() - start) * 1000))`
         - On `ImportError`: log "SOCKS5 proxy health check failed: install httpx[socks]", return `("dead", None)`
         - On any other `Exception`: log warning, return `("dead", None)`

    3. Create `app/services/service_credential_service.py`:
       - `import json` + imports for `crypto_service`, `ServiceCredential`, `select` from sqlalchemy
       - `ENCRYPTED_FIELDS = {"xmlproxy": ["key"], "rucaptcha": ["key"]}` — lists which JSON keys get Fernet-encrypted per service
       - `def save_credential_sync(db: Session, service_name: str, data: dict) -> ServiceCredential`:
         - Deep-copy data; for each field in `ENCRYPTED_FIELDS.get(service_name, [])`, call `crypto_service.encrypt(data[field])`
         - `json.dumps` the result into `credential_data`
         - Upsert: query existing by service_name, update credential_data if exists, else insert new
         - `db.commit()`
       - `def get_credential_sync(db: Session, service_name: str) -> dict | None`:
         - Query `ServiceCredential` by `service_name`
         - If not found or `credential_data` is None, return None
         - `json.loads(credential_data)`, decrypt fields listed in `ENCRYPTED_FIELDS`, return dict
