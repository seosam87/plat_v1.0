<task id="02">
<title>Create app/models/architecture.py with SitemapEntry and PageLink</title>
<read_first>
- app/models/crawl.py
- app/database.py
</read_first>
<action>
Create `app/models/architecture.py`:

**SitemapEntry** — stores sitemap.xml comparison results:
- `id`: UUID PK
- `site_id`: UUID FK → sites.id CASCADE
- `url`: String(2000), not null
- `in_sitemap`: Boolean, default True — URL found in sitemap.xml
- `in_crawl`: Boolean, default False — URL found in crawl/SF data
- `status`: String(20), nullable — 'orphan' (in crawl, not sitemap), 'missing' (in sitemap, not crawl), 'ok' (both)
- `last_modified`: DateTime, nullable — from sitemap <lastmod>
- `created_at`: DateTime(timezone=True)
- `__table_args__`: UniqueConstraint('site_id', 'url', name='uq_sitemap_entry_site_url')

**PageLink** — internal link graph:
- `id`: UUID PK
- `site_id`: UUID FK → sites.id CASCADE
- `crawl_job_id`: UUID FK → crawl_jobs.id CASCADE
- `source_url`: String(2000), not null — page containing the link
- `target_url`: String(2000), not null — link destination
- `anchor_text`: String(500), nullable
- `created_at`: DateTime(timezone=True)
- `__table_args__`: Index on (site_id, crawl_job_id, source_url)

Table names: `sitemap_entries`, `page_links`.
</action>
