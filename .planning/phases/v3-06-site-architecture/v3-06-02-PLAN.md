### SF Import (async)

**`async def import_sf_data(db, site_id, file_path) -> dict`**
1. Parse SF file via `parse_screaming_frog()`
2. For each page in result: upsert into Page model with source='sf_import'
3. Update inlinks_count from SF inlinks data
4. Return `{imported: N, tab_type: str}`

### Sitemap comparison

**`def parse_sitemap_xml(content: str) -> list[dict]`** (pure)
Parse sitemap.xml content. Extract `{url, lastmod}` entries.
Handle both plain sitemaps and sitemap index files (recursive).

**`async def fetch_sitemap(site_url: str) -> str | None`**
HTTP GET `{site_url}/sitemap.xml`. Returns content string or None.

**`async def compare_sitemap(db, site_id, sitemap_urls: list[dict]) -> dict`**
1. Get all crawled page URLs for site
2. Compare with sitemap URLs:
   - `orphan`: in crawl, not in sitemap
   - `missing`: in sitemap, not in crawl
   - `ok`: in both
3. Upsert SitemapEntry records
4. Return `{total_sitemap, total_crawl, orphan: N, missing: N, ok: N}`

### URL Tree (pure)

**`def build_url_tree(urls: list[str]) -> dict`**
Build a nested tree from URL path segments:
```python
{
    "name": "/",
    "children": [
        {"name": "uslugi", "url": "/uslugi/", "children": [
            {"name": "seo", "url": "/uslugi/seo/", "children": [...]},
        ]},
        {"name": "blog", "url": "/blog/", "children": [...]},
    ]
}
```
D3.js-compatible format. Each node has: name, full_url (if page exists), page_count (total descendants), children.

### Architecture Role Detection (async)

**`async def detect_architecture_roles(db, site_id, crawl_job_id) -> int`**
Auto-detect roles based on heuristics:
- `page_type == 'landing'` + high inlinks → `pillar`
- `page_type == 'product'` or URL contains /uslugi/, /services/ → `service`
- URL contains sub-path of a service → `subservice`
- `page_type == 'article'` or `content_type == 'informational'` → `article`
- URL contains /otzyvy/, /reviews/, /cases/, /sertifikaty/ → `authority`
- URL contains /promo/, /akcii/, /skidki/ → `trigger`
- High inlinks_count + `content_type == 'informational'` + many external links → `link_accelerator`
- Default → `unknown`

Return count of pages classified.

**`async def update_page_role(db, page_id, role) -> None`** — manual override.

### Inlinks Diff (sync for Celery)

**`def compute_inlinks_diff(old_links: list[dict], new_links: list[dict]) -> dict`** (pure)
Compare two sets of PageLink records:
- `added`: links in new but not old
- `removed`: links in old but not new
- `unchanged`: in both
Return `{added: [{source_url, target_url, anchor}], removed: [...], added_count, removed_count}`.

**`def save_page_links(db, site_id, crawl_job_id, links: list[dict]) -> int`** (sync)
Bulk insert PageLink records from crawl data. Return count.
