# Plan v3-06-03: Router, D3.js tree UI, and integration

<objective>
Create architecture router with endpoints for SF import, sitemap comparison, URL tree data, architecture role management, and inlinks diff. Build UI with D3.js collapsible tree, sitemap orphan/missing table, architecture role map, and inlinks diff view. Add "Архитектура" button to site detail.
</objective>

<tasks>

<task id="01">
<title>Create architecture router</title>
<read_first>
- app/routers/gap.py (router pattern with file upload)
- app/services/architecture_service.py
</read_first>
<action>
Create `app/routers/architecture.py`:

Router: `prefix="/architecture"`, tags `["architecture"]`

**Page:**
1. `GET /{site_id}` — Main architecture page (HTML)

**SF Import:**
2. `POST /{site_id}/import-sf` — Upload SF CSV/XLSX file. Multipart form.

**Sitemap:**
3. `POST /{site_id}/sitemap/fetch` — Fetch sitemap.xml from site URL
4. `POST /{site_id}/sitemap/upload` — Upload sitemap.xml file
5. `GET /{site_id}/sitemap/results` — Comparison results (orphan/missing/ok list)

**Tree:**
6. `GET /{site_id}/tree` — URL tree data (JSON, D3.js format)

**Architecture roles:**
7. `POST /{site_id}/detect-roles` — Auto-detect roles
8. `PUT /{site_id}/pages/{page_id}/role` — Manual role override. Body: `{role}`.
9. `GET /{site_id}/roles` — Pages grouped by architecture role (JSON)

**Inlinks:**
10. `GET /{site_id}/inlinks-diff` — Diff between last two crawls. Query: `crawl_a`, `crawl_b`.

All endpoints `require_admin`.
</action>

</task>

<task id="02">
<title>Create architecture page template with D3.js tree</title>
<read_first>
- app/templates/analytics/index.html (UI pattern)
</read_first>
<action>
Create `app/templates/architecture/index.html`:

Extends `base.html`. Sections:

**1. Header:** "Архитектура сайта: {site.name}"

**2. Import section** (card):
- Two columns: "Импорт Screaming Frog" (file upload) + "Sitemap.xml" (fetch button + file upload)
- Status message area

**3. URL Tree** (card):
- D3.js collapsible tree loaded from `/architecture/{site_id}/tree`
- Use d3.hierarchy + d3.tree layout
- Nodes colored by architecture_role (pillar=purple, service=blue, article=green, trigger=orange, authority=teal, link_accelerator=pink, unknown=gray)
- Click node → show page details
- CDN: d3.js v7

**4. Architecture Map** (card):
- Table grouped by role: Pillar → Service → Subservice → Article → etc.
- Each row: URL, title, role (editable select), inlinks count, page_type
- Role select → PUT to update

**5. Sitemap Comparison** (card):
- Summary: N orphan, N missing, N ok
- Table with filter (all/orphan/missing): URL, Status badge, Last Modified
- Color: orphan=red, missing=yellow, ok=green

**6. Inlinks Diff** (card):
- Select two crawls (dropdown)
- Table: Source URL → Target URL | Anchor | Status (added=green, removed=red)
- Summary: +N added, -N removed

All copy in Russian.
</action>
