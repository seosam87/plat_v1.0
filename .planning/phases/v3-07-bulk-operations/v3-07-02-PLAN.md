# Plan v3-07-02: Bulk router and UI

<objective>
Create bulk operations router and UI page with keyword table (select-all, batch actions), group/cluster/target_url assignment controls, delete confirmation, CSV/XLSX export buttons, and file import with merge mode selector.
</objective>

<tasks>

<task id="01">
<title>Create bulk router</title>
<read_first>
- app/routers/analytics.py (filter pattern)
- app/services/bulk_service.py
</read_first>
<action>
Create `app/routers/bulk.py`:

Router: `prefix="/bulk"`, tags `["bulk"]`

1. `GET /{site_id}` — Bulk operations page (HTML). Loads keywords via filter + groups + clusters.
2. `POST /{site_id}/move-group` — Batch move to group. Body: `{keyword_ids, group_id}`.
3. `POST /{site_id}/move-cluster` — Batch move to cluster. Body: `{keyword_ids, cluster_id}`.
4. `POST /{site_id}/assign-url` — Batch assign target_url. Body: `{keyword_ids, target_url}`.
5. `POST /{site_id}/delete` — Batch delete. Body: `{keyword_ids}`.
6. `GET /{site_id}/export` — Export filtered keywords. Query: `format=csv|xlsx`, plus filter params. Returns file download.
7. `POST /{site_id}/import` — Import file. Multipart form: file + on_duplicate field.

All `require_admin`.
</action>
<acceptance_criteria>
- [ ] `python -c "from app.routers.bulk import router"` succeeds
- [ ] `grep -c "@router" app/routers/bulk.py` returns at least 6
</acceptance_criteria>
</task>

<task id="02">

**1. Header:** "Массовые операции: {site.name}"

- Columns: ☐ | Фраза | Частотность | Позиция | Группа | Кластер | Target URL
