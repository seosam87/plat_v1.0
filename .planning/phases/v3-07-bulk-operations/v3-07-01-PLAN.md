# Plan v3-07-01: Bulk service

<objective>
Create bulk operations service: batch move keywords between groups/clusters, batch assign target_url, batch delete, filtered export (CSV + XLSX), import with merge and audit logging.
</objective>

<action>
Create `app/services/bulk_service.py`:

### Batch operations (async)

**`async def bulk_move_to_group(db, keyword_ids: list[UUID], group_id: UUID | None) -> int`**
Set group_id for all keywords. group_id=None removes from group. Return count updated.

**`async def bulk_move_to_cluster(db, keyword_ids: list[UUID], cluster_id: UUID | None) -> int`**
Set cluster_id for all keywords. Return count.

**`async def bulk_assign_target_url(db, keyword_ids: list[UUID], target_url: str) -> int`**
Set target_url for all keywords. Return count.

**`async def bulk_delete(db, keyword_ids: list[UUID]) -> int`**
Delete keywords by IDs. Return count deleted.

**`async def bulk_delete_by_filter(db, site_id: UUID, *, group_id=None, cluster_id=None, search=None) -> int`**
Delete keywords matching filter criteria. Return count.

### Export (pure + async)

**`async def export_keywords_csv(db, site_id: UUID, keyword_ids: list[UUID] | None = None, **filters) -> str`**
If keyword_ids provided, export those. Otherwise use filters (same as analytics filter_keywords).
CSV columns: Phrase, Frequency, Region, Engine, Group, Cluster, Target URL, Position, Delta.

**`async def export_keywords_xlsx(db, site_id: UUID, keyword_ids: list[UUID] | None = None, **filters) -> bytes`**
Same as CSV but as XLSX bytes using openpyxl.

### Import with audit log (async)

**`async def import_keywords_with_log(db, site_id: UUID, file_path: str, on_duplicate: str = "skip", user_id: UUID | None = None) -> dict`**
1. Parse file (CSV/XLSX) via existing parsers
2. Call existing `bulk_add_keywords(db, site_id, keywords, on_duplicate)`
3. Write audit log entry via `audit_service.log_action()`:
   - action: "bulk_keyword_import"
   - entity_type: "keywords"
   - entity_id: str(site_id)
   - detail: `{"file": filename, "on_duplicate": on_duplicate, "total": N, "added": N, "updated": N, "skipped": N}`
4. Return `{total, added, updated, skipped}`
</action>
