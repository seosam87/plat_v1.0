       Define these endpoints (all details in Task 2 action for template-related endpoints):

       **GET /ui/crm/clients** — Client list page (per D-12, D-13, D-14, D-15):
       - Auth: `require_any_authenticated`
       - Query params: `search: str | None`, `manager_id: uuid.UUID | None`, `page: int = 1`, `partial: str | None` (for HTMX)
       - Call `client_service.list_clients(db, search, manager_id, page, 25)`
       - Also fetch managers list: `select(User).where(User.role.in_([UserRole.admin, UserRole.manager]))` for the filter dropdown
       - If `partial == "true"`: return `templates.TemplateResponse(request, "crm/index.html", {...}, block_name="table_body")` or render just the tbody + pagination partial
       - Else: return full page `templates.TemplateResponse(request, "crm/index.html", {...})`
       - Template context: `{"clients": clients, "total": total, "page": page, "page_size": 25, "search": search or "", "manager_id": manager_id, "managers": managers}`

       **GET /ui/crm/clients/new** — Create modal fragment (per D-03):
       - Auth: `require_manager_or_above`
       - Fetch managers list for dropdown
       - Return `templates.TemplateResponse(request, "crm/_modal_client.html", {"client": None, "managers": managers})`

       **GET /ui/crm/clients/{client_id}/edit** — Edit modal fragment (per D-03):
       - Auth: `require_manager_or_above`
       - Fetch client + managers list
       - Return `templates.TemplateResponse(request, "crm/_modal_client.html", {"client": client, "managers": managers})`

       **POST /ui/crm/clients** — Create client (per CRM-01, CRM-02):
       - Auth: `require_manager_or_above`
       - Form fields: `company_name: str = Form(...)`, `legal_name: str = Form("")`, `inn: str = Form("")`, `kpp: str = Form("")`, `phone: str = Form("")`, `email: str = Form("")`, `manager_id: str = Form("")`, `notes: str = Form("")`
       - Convert empty strings to None for nullable fields
       - Parse manager_id to UUID or None
       - Call `client_service.create_client(db, data_dict)`
       - Call `log_action(db, user_id=current_user.id, action="create", entity_type="client", entity_id=str(client.id), details=f"Created client {client.company_name}")`
       - Return `HTMLResponse` with HX-Trigger header `{"showToast": "Клиент создан", "closeModal": true, "refreshClientList": true}` or redirect

       **PUT /ui/crm/clients/{client_id}** — Update client:
       - Same form fields as POST
       - Call `client_service.update_client(db, client_id, data_dict)`
       - Call `log_action` with action="update"
       - Return similar HX-Trigger response

       **DELETE /ui/crm/clients/{client_id}** — Soft delete client:
       - Auth: `require_manager_or_above`
       - Call `client_service.delete_client(db, client_id)` (soft delete)
       - Call `log_action` with action="delete"
       - Return HX-Trigger with refreshClientList

       NOTE: Contact, interaction, and site-linking endpoints will be added in Plans 03 and 04. This plan covers only list + CRUD + modal endpoints.
