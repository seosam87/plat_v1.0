    Add the following endpoints to the existing `app/routers/crm.py` (created in Plan 02):

    **GET /ui/crm/clients/{client_id}** — Client detail page (per D-01, D-02, CRM-07):
    - Auth: `require_any_authenticated`
    - Fetch client via `client_service.get_client(db, client_id)`, raise 404 if None
    - Fetch contacts: `select(ClientContact).where(ClientContact.client_id == client_id).order_by(ClientContact.name)`
    - Fetch interactions: `client_service.list_interactions(db, client_id, page=1, page_size=50)`
    - Fetch linked sites: `select(Site).where(Site.client_id == client_id, Site.is_active == True)`
    - Fetch open task count: `client_service.get_open_task_count_for_client(db, client_id)`
    - Fetch site count: `select(func.count(Site.id)).where(Site.client_id == client_id, Site.is_active == True)`
    - Fetch manager: if `client.manager_id`, `select(User).where(User.id == client.manager_id)`
    - Return `templates.TemplateResponse(request, "crm/detail.html", {"client": client, "contacts": contacts, "interactions": interactions, "sites": sites, "open_task_count": open_task_count, "site_count": site_count, "manager": manager, "current_user": current_user})`

    **Contact endpoints (per D-07, CRM-04):**

    POST /ui/crm/clients/{client_id}/contacts — Create contact:
    - Auth: `require_manager_or_above`
    - Form fields: `name: str = Form(...)`, `phone: str = Form("")`, `email: str = Form("")`, `role: str = Form("")`, `telegram_username: str = Form("")`, `notes: str = Form("")`
    - Convert empty strings to None
    - Call `client_service.create_contact(db, client_id, data_dict)`
    - Call `log_action(db, user_id=current_user.id, action="create", entity_type="client_contact", entity_id=str(contact.id), details=f"Added contact {contact.name}")`
    - Return `templates.TemplateResponse(request, "crm/_contact_row.html", {"contact": contact, "client_id": client_id, "current_user": current_user})`

    GET /ui/crm/clients/{client_id}/contacts/{contact_id}/edit — Contact edit mode (per D-07):
    - Auth: `require_manager_or_above`
    - Fetch contact, return `templates.TemplateResponse(request, "crm/_contact_edit.html", {"contact": contact, "client_id": client_id})`

    GET /ui/crm/clients/{client_id}/contacts/{contact_id} — Contact read mode (cancel edit):
    - Auth: `require_any_authenticated`
    - Fetch contact, return `templates.TemplateResponse(request, "crm/_contact_row.html", {"contact": contact, "client_id": client_id, "current_user": current_user})`

    PUT /ui/crm/clients/{client_id}/contacts/{contact_id} — Update contact:
    - Auth: `require_manager_or_above`
    - Same form fields as create
    - Call `client_service.update_contact(db, contact_id, client_id, data_dict)`
    - Return `_contact_row.html` partial (back to read mode)

    DELETE /ui/crm/clients/{client_id}/contacts/{contact_id} — Delete contact:
    - Auth: `require_manager_or_above`
    - Call `client_service.delete_contact(db, contact_id)`
    - Return `HTMLResponse("")` (row removed from DOM)

    **Interaction endpoints (per D-08, D-09, D-10, D-11, CRM-05):**

    POST /ui/crm/clients/{client_id}/interactions — Create interaction:
    - Auth: `require_manager_or_above`
    - Form field: `note: str = Form(...)`
    - Call `client_service.create_interaction(db, client_id, current_user.id, note)`
    - Return `templates.TemplateResponse(request, "crm/_interaction_entry.html", {"interaction": interaction, "author": current_user, "current_user": current_user})`

    PUT /ui/crm/clients/{client_id}/interactions/{interaction_id} — Update interaction:
    - Auth: `require_manager_or_above`
    - Check: current_user.id == interaction.author_id OR current_user.role == UserRole.admin (per D-10)
    - Form field: `note: str = Form(...)`
    - Call `client_service.update_interaction(db, interaction_id, note)`
    - Return `_interaction_entry.html` partial

    DELETE /ui/crm/clients/{client_id}/interactions/{interaction_id} — Delete interaction:
    - Auth: `require_manager_or_above`
    - Same ownership check as update (per D-10)
    - Call `client_service.delete_interaction(db, interaction_id)`
    - Return `HTMLResponse("")`
