**Step 2: Create `app/routers/tools.py`**

Minimal FastAPI router with a single GET endpoint. No path parameters (avoids PARAM_MAP issues — Pitfall 3). Use `/ui/tools/` prefix so it is auto-discovered by smoke tests via `/ui/` prefix (Pitfall 5).

```python
from fastapi import APIRouter, Request
from fastapi.responses import HTMLResponse

router = APIRouter(prefix="/ui/tools", tags=["tools"])


@router.get("/", response_class=HTMLResponse, name="tools_index")
async def tools_index(request: Request):
    return request.app.state.templates.TemplateResponse(
        "tools/index.html",
        {"request": request},
    )
```

Follow the pattern used in other routers (e.g., keyword_suggest.py) for template response. Check how `templates` is accessed — it may be `request.app.state.templates` or `templates` imported from a module. Read an existing simple router to confirm the exact pattern.
