| sm | 8px | Gap between toast items, inline padding within health metric row |

## Typography

All sizes use `system-ui, sans-serif`. No custom web fonts — reduces mobile page weight.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px (0.875rem) | 400 | 1.5 | Digest item text, health metric labels, form field values |
| Label | 12px (0.75rem) | 400 | 1.4 | Section sub-labels, bottom nav tab labels, metadata (site name, date) |
| Heading | 20px (1.25rem) | 600 | 1.2 | Page title ("Дайджест — 10 апр", "Здоровье сайта") |
| Section heading | 16px (1rem) | 600 | 1.2 | Block titles within digest ("Изменения позиций", "Ошибки краулера") |

Note: Only 4 sizes declared. Body (14px) and Label (12px) share weight 400; Heading (20px) and Section heading (16px) share weight 600.

## Interaction Contract

### Digest Page (`/m/digest`)

1. **Page load:** Full page render via FastAPI GET. No HTMX partial on initial load.
2. **Tap on digest item:** Navigate to deep link (anchor tag). Destination: desktop `/ui/` pages (Phase 28–31 will update these to mobile). No HTMX.
3. **Tap "Подробнее →":** Same as item tap — single `<a>` element.
4. **Empty block:** Rendered server-side as static `<p>` text. No JS interaction.

### Health Card (`/m/health/{site_id}`)

1. **Page load:** Full page render via FastAPI GET. All 6 metrics rendered server-side.
2. **Tap "Запустить краулинг":**
   - Button has `hx-post="/m/health/{site_id}/crawl"`, `hx-swap="none"`.
   - On success: `hx-on::after-request="if(event.detail.successful) showToast('Краулинг запущен', 'success')"`.
   - On error: global `htmx:responseError` handler in `base_mobile.html` shows error toast automatically.
   - Button enters disabled state during request (`hx-indicator` or CSS `pointer-events:none`).
3. **Tap "Создать задачу" (collapsed state):**
   - Button has `hx-get="/m/health/{site_id}/task-form?url={error_url|urlencode}"`, `hx-target="#task-form-slot"`, `hx-swap="innerHTML"`.
   - Expands inline form below the metric row. Original button hides after expansion.
4. **Inline task form — "Сохранить задачу":**
   - Form has `hx-post="/m/health/{site_id}/tasks"`, `hx-swap="outerHTML"`, `hx-target="closest form"`.
   - On success (HTTP 201): form replaced with empty string. `hx-on::after-request` calls `showToast('Задача создана', 'success')`.
   - On error: global error handler shows toast.
5. **Inline task form — "Отмена":**
   - `hx-get="/m/health/{site_id}/task-form/cancel"` returns empty string, `hx-target="#task-form-slot"`, `hx-swap="innerHTML"`. OR: plain JS `document.getElementById('task-form-slot').innerHTML = ''`. Use JS approach — no extra endpoint.
6. **HTMX event syntax:** Always `hx-on::after-request` (double colon — HTMX 2.0.3 is loaded).

### Touch Target Rules

- All tappable rows in digest: `min-height: 44px`, full-width tap area (`display: block` on `<a>`).
- All action buttons: `min-height: 44px`, `padding: 12px 16px`.
- Bottom nav tabs: `min-height: 44px` (already enforced in base_mobile.html).
- Inline form fields: `height: 44px` minimum on mobile.

## Layout Contract

### /m/digest layout

```
[version-switch strip]         ← inherited from base_mobile.html
[main p-4]
  [H1 "Дайджест — {date}"]     ← 20px/600
  [Section: Изменения позиций] ← mb-6
    [H2 "Изменения позиций"]   ← 16px/600, mb-2
    [list of ≤5 items]         ← each item: min-h-44px, border-b, flex row
      [status indicator dot]   ← 8px, green/red
      [keyword + site_name]    ← 14px/400, flex-1
      [delta badge]            ← 12px/600, green/red text
      [chevron right →]        ← 16px icon, tap target extends full row
    [empty state <p>]          ← 14px/400, text-gray-500, if no items
  [Section: Ошибки краулера]   ← same pattern
  [Section: Алерты]            ← same pattern
  [Section: Просроченные задачи] ← same pattern
[bottom-nav]                   ← inherited, fixed
[toast-container]              ← inherited, fixed
```

### /m/health/{site_id} layout

```
[version-switch strip]
[main p-4]
  [H1 "Здоровье: {site_name}"]  ← 20px/600, mb-4
  [metric list]                  ← vertical stack, gap-2
    [metric row × 6]             ← each: white card, rounded-lg, p-4, min-h-44px
      [status dot 8px]           ← green/yellow/red/grey
      [metric label]             ← 14px/400, flex-1
      [metric value]             ← 14px/600 or 12px, right-aligned
  [action row]                   ← flex gap-2, mt-6
    [Запустить краулинг btn]     ← accent bg, white text, flex-1, min-h-44px
    [Создать задачу btn]         ← white bg, accent border+text, flex-1, min-h-44px
  [#task-form-slot]              ← empty div, filled by HTMX on task btn tap
    [inline form, if expanded]   ← white card, p-4, rounded-lg
      [title input]              ← full-width, min-h-44px
      [priority select]          ← full-width, min-h-44px
      [Сохранить / Отмена btns]  ← flex row, gap-2
[bottom-nav]
[toast-container]
```
