## Decisions Made
- Used `window.location.assign` (not HTMX boost) for `open` op navigation — HTMX swap does NOT fire DOMContentLoaded, so the sessionStorage resume handler would never run
- Launcher button uses `btn.style.cssText` inline styles — Tailwind CDN JIT does not guarantee class availability for dynamically inserted elements
- `text=` and `label=` locators log a warning and return null (step becomes centered modal) — Shepherd requires CSS selectors and accessible-name tree walking is out of scope
