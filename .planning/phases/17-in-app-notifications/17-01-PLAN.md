    Per D-01 + specifics block, create `app/services/notifications.py`:
      ```python
      from typing import Literal
      from uuid import UUID
      from sqlalchemy.ext.asyncio import AsyncSession
      from app.models.notification import Notification

      async def notify(
          db: AsyncSession,
          user_id: UUID,
          kind: str,
          title: str,
          body: str,
          link_url: str,
          site_id: UUID | None = None,
          severity: Literal["info", "warning", "error"] = "info",
      ) -> Notification:
          n = Notification(
              user_id=user_id, kind=kind, title=title, body=body,
              link_url=link_url, site_id=site_id, severity=severity,
          )
          db.add(n)
          await db.flush()
          return n
      ```
    Create `app/tasks/notification_tasks.py` with Celery task `cleanup_old_notifications` that:

      - Logs deleted row count via loguru
      - Returns the count
