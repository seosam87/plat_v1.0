  <action>
    Create `app/routers/notifications.py` with `APIRouter(prefix="/notifications")`. Endpoints:

    1. `GET /notifications/bell` (HTMX fragment)
       - Returns `notifications/_bell.html` with context: `unread_count`, `has_unread_error` (bool — true if any unread Notification with severity='error').
       - Per D-07: badge red if has_unread_error else default.
       - Used by 30s poll (D-06, silent).

    2. `GET /notifications/dropdown` (HTMX fragment)
       - Returns `notifications/_dropdown.html` with last 10 notifications for current_user, ordered by created_at DESC, flat list (D-04).
       - On render, atomically mark those visible entries as is_read=True (D-05). Implement as a single UPDATE with `WHERE id IN (...)`.

    3. `GET /notifications` (full page)
       - Returns `notifications/index.html`.
       - Query params: `site_id` (UUID or empty), `kind` (str or empty), `read_state` (`all`|`unread`, default `all`), `page` (int, default 1).
       - Pagination 50/page (D-09).
       - Build distinct sites + distinct kinds for the filter bar (only those present in user's history).
       - Group results by `kind` (D-08); within each group order by created_at DESC.

    4. `POST /notifications/mark-all-read` (HTMX)
       - Marks ALL of current_user's notifications is_read=True. Returns updated `_list.html` fragment respecting current filters (read query params).

    5. `POST /notifications/{id}/dismiss` (HTMX)
       - Hard-delete (per phase boundary). Verify ownership (user_id == current_user.id). Returns 204 or empty fragment.

    Use existing `current_user` dependency. All endpoints require auth.

    Register router in `app/main.py`: `app.include_router(notifications.router)`.

    Tests in `tests/routers/test_notifications.py` (httpx AsyncClient):
      - test_bell_fragment_unread_count: seed 3 unread, assert response contains "3"
      - test_bell_red_when_error: seed 1 unread error, assert response has red badge marker class
      - test_dropdown_marks_read: seed 5 unread, GET dropdown, assert all 5 now is_read=True
      - test_dropdown_limit_10: seed 15, assert response shows only 10
      - test_index_filter_by_kind: seed 2 crawl + 3 pdf, GET ?kind=pdf.ready, assert 3
      - test_index_filter_by_site: filter by site_id
      - test_index_filter_unread: seed mix, GET ?read_state=unread, assert only unread shown
      - test_mark_all_read: seed 5 unread, POST mark-all-read, assert all is_read=True
      - test_dismiss_deletes: POST dismiss, assert row gone from DB
      - test_dismiss_other_user_forbidden: 403/404
  </action>
